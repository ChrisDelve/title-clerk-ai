==================================================
PROCEDURE SUMMARY
==================================================

This file governs:
- Florida vessel registration fees
- Vessel class fee structure
- Recreational vessel reduced registration fees
- EPIRB reduced fee eligibility
- PLB reduced fee eligibility
- Optional county vessel fees
- Dealer vessel registration fees
- Commercial vessel additional license fees
- Vessel registration class determination by length

Applies to:
- Recreational vessels
- Boats
- Dealer vessel registrations
- Commercial vessel registrations
- Non-resident commercial vessels
- Alien commercial vessel registrations

==================================================
VERY IMPORTANT CORE CONCEPT
==================================================

FEES-04 is a:
VESSEL REGISTRATION FEE CLASSIFICATION CHART.

The chart determines:
- vessel registration class
- vessel registration fee
- reduced EPIRB/PLB fee eligibility
- optional county fee amounts

AI must:
1. Determine vessel length
2. Determine vessel class
3. Determine standard fee
4. Determine reduced EPIRB/PLB fee eligibility
5. Determine optional county fee applicability
6. Determine commercial/non-resident applicability

==================================================
STATUTORY AUTHORITY
==================================================

Section:
328.72(18), Florida Statutes

Provides:
Reduced vessel registration fees for:
- recreational vessels equipped with EPIRB
OR
- recreational vessel owners possessing PLB

==================================================
EPIRB / PLB REDUCED FEE LOGIC
==================================================

Reduced fee eligibility applies when:
- recreational vessel equipped with EPIRB
OR
- owner possesses PLB

EPIRB:
Emergency Position Indicating Radio Beacon

PLB:
Personal Locator Beacon

AI should:
verify qualifying safety equipment before applying reduced fee.

IMPORTANT IMPLEMENTATION NOTES
==================================================

Chart totals DO NOT include:
- service fee
- aquatic plate fee
- Save the Manatee fee

separate:
BASE REGISTRATION FEES
from:
MANDATORY ADDITIONAL FEES.
