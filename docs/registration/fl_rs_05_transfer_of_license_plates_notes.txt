==================================================
VERY IMPORTANT CORE RULE
==================================================

License plates:
May be transferred:
To replacement vehicles:
Under qualifying ownership conditions.

==================================================
TRANSFER FEE EXEMPT CLASS CODES
==================================================

Transfer fee exempt:
When BOTH original and replacement vehicles:
Class codes:
01
31
37
96
97

==================================================
VERY IMPORTANT TRANSFER FEE RULE
==================================================

Transfer fee required:
Unless:
- Exempt class code
- Plate expired and renewed same transaction
- Plate transferred to same vehicle

==================================================
VERY IMPORTANT SAME VEHICLE RULE
==================================================

Transfer fee NOT required:
When transferring:
To same vehicle.

==================================================
LEASE BUYOUT RULE
==================================================

Example:
Lessee purchases lease vehicle:
Transfer fee not required.

==================================================
VERY IMPORTANT REGISTRATION CREDIT RULE
==================================================

Credit allowed:
From previously paid registration tax:
Toward new registration.

==================================================
VERY IMPORTANT RS-10 RULE
==================================================

Mobile home decal transfers:
Governed by:
RS-10.

==================================================
ETR SYSTEM RULES
==================================================

ETR:
Electronic Temporary Registration system.

==================================================
VERY IMPORTANT ETR PURPOSE RULE
==================================================

ETR allows dealers:
To electronically transfer:
License plates:
At time of sale.

==================================================
VERY IMPORTANT METAL PLATE RULE
==================================================

Electronic temporary transfer:
Only required:
When metal plate physically affixed:
To replacement vehicle.

==================================================
VERY IMPORTANT DEALER TRANSFER RULE
==================================================

Dealer enters:
Customer and vehicle information:
Into ETR system.

==================================================
VERY IMPORTANT SYSTEM VERIFICATION RULE
==================================================

ETR verifies:
Against:
Motor vehicle issuance system.

==================================================
VERY IMPORTANT PLATE AFFIX RULE
==================================================

Dealer affixes:
Customer metal plate:
To replacement vehicle.

==================================================
ETR FAILURE RULE
==================================================

If ETR denies transfer:
Dealer may:
- Issue temporary print-on-demand plate
OR
- Visit tax collector office.

==================================================
VERY IMPORTANT ETR OUTAGE RULE
==================================================

During ETR outage:
Dealer may issue:
Pre-printed temporary plate.

==================================================
VERY IMPORTANT ONE BUSINESS DAY RULE
==================================================

Pre-printed temporary plate:
Must be electronically reported:
Within one business day.

==================================================
VERY IMPORTANT RS-31 RULE
==================================================

Temporary plate issuance:
Governed by:
RS-31.

==================================================
VERY IMPORTANT NO PLATE RULE
==================================================

If customer lacks metal plate:
Dealer must:
- Issue temporary plate
OR
- Visit tax collector office.

==================================================
VERY IMPORTANT ONLY ONE PLATE RULE
==================================================

Only one valid plate:
May be displayed:
At any time.

==================================================
VERY IMPORTANT TEMP + METAL RULE
==================================================

Regular plate and temporary plate:
Cannot both be displayed.

==================================================
EFS RULES
==================================================

EFS:
Electronic Filing System.

==================================================
VERY IMPORTANT EFS PURPOSE RULE
==================================================

EFS allows dealer:
To:
- Transfer plate
- Print registration
- Transfer registration electronically.

==================================================
NON-ETR / NON-EFS DEALER RULE
==================================================

Dealers without ETR/EFS:
Must visit:
Tax collector office.

==================================================
VERY IMPORTANT DEALER REQUIREMENT RULE
==================================================

Non-ETR/EFS dealer:
Cannot allow customer:
To leave lot:
Without proper transaction completed.

==================================================
ALLOWED NON-ETR TRANSACTIONS
==================================================

Allowed:
- Temporary plate issuance
- Temporary transfer
- Permanent registration issuance

==================================================
NAME CHANGE RULES
==================================================

Legal name changes:
Require:
Motor vehicle issuance system update.

==================================================
VERY IMPORTANT SYSTEM UPDATE RULE
==================================================

After customer maintenance update:
All DMS records:
Reflect new name.

==================================================
VERY IMPORTANT TITLE MODIFICATION RULE
==================================================

Customer may optionally:
Modify title:
To new legal name.

==================================================
INDIVIDUAL NAME CHANGE RULE
==================================================

Florida credential:
Must reflect:
New name.

==================================================
VERY IMPORTANT NON-FLORIDA INDIVIDUAL RULE
==================================================

If no Florida credential:
Clerk may request:
Additional verification.

==================================================
CORPORATE NAME CHANGE RULE
==================================================

Corporation changing only name:
May retain:
Same plate/registration.

==================================================
VERY IMPORTANT CORPORATE AFFIDAVIT RULE
==================================================

Require:
Corporate name change affidavit:
Filed with Florida Department of State.

==================================================
VERY IMPORTANT LLC CONVERSION RULE
==================================================

Certificate of Conversion:
May transfer:
Plate/registration:
From INC to LLC.

==================================================
VERY IMPORTANT LEGAL OPINION RULE
==================================================

FLHSMV legal opinion:
Allows:
Corporate conversion transfers:
Using Department of State conversion filing.

==================================================
FICTITIOUS NAME RULE
==================================================

Fictitious business name change:
Requires:
Registered fictitious name filing.

==================================================
VERY IMPORTANT DBA RULE
==================================================

Fictitious name:
Means:
Doing business as:
Different from legal name.

==================================================
TRUST / TRUSTEE RULES
==================================================

Transfer from individual:
To trust:
Normally requires:
New plate.

==================================================
VERY IMPORTANT ENTITY RULE
==================================================

Trust:
Considered:
Different entity than natural person.

==================================================
VERY IMPORTANT BENEFICIAL USE RULE
==================================================

Plate transfer allowed:
If:
Vehicle titled in trust:
But registered in individual's name:
As registrant.

==================================================
VERY IMPORTANT TRUST AFFIDAVIT RULE
==================================================

Require affidavit:
Individual has:
Beneficial use of vehicle.

==================================================
VERY IMPORTANT TRUST FLAG RULE
==================================================

Customer detail maintenance:
Must mark:
TRUST flag with "X".

==================================================
SURVIVING SPOUSE RULES
==================================================

Surviving spouse:
May transfer:
Decedent's plate.

==================================================
VERY IMPORTANT DEATH CERTIFICATE RULE
==================================================

Require:
Death certificate
OR
Death File Inquiry.

==================================================
VERY IMPORTANT DEATH FILE RULE
==================================================

Death File Inquiry:
Accessible:
Through motor vehicle issuance system.

==================================================
VERY IMPORTANT CONFIDENTIALITY RULE
==================================================

Death File Inquiry printout:
Confidential:
Cannot be given to customer.

==================================================
VERY IMPORTANT QUALIFYING CONDITION RULE
==================================================

Special qualifying plate:
Cannot transfer:
Unless surviving spouse also qualifies.

==================================================
VERY IMPORTANT NEW DECAL RULE
==================================================

New validation decal:
Required:
To match surviving spouse birthday cycle.

==================================================
VERY IMPORTANT SPOUSE CREDIT RULE
==================================================

Surviving spouse:
May use registration credit:
Toward newly purchased vehicle.

==================================================
VERY IMPORTANT REGISTRATION TRANSFER OWNERSHIP RULE
==================================================

Spouse transfer:
Must first process:
Registration Transfer Ownership transaction.

==================================================
GENERAL INFORMATION RULES
==================================================

Seller should retain:
License plate:
For future transfer/credit.

==================================================
VERY IMPORTANT NO REFUND RULE
==================================================

No refund:
If replacement vehicle not purchased.

==================================================
VERY IMPORTANT SURRENDER RULE
==================================================

If not transferring plate:
Customer should:
- Cancel insurance
- Surrender plate

==================================================
VERY IMPORTANT INSURANCE RULE
==================================================

Proof of motor vehicle insurance:
Required:
When transferring plate.

==================================================
VERY IMPORTANT NON-OPERATIONAL RULE
==================================================

Plate may transfer:
Even if original vehicle not sold.

==================================================
VERY IMPORTANT RS-30 RULE
==================================================

Require affidavit:
Vehicle:
- Not operational
- In storage
OR
- Not operated on roads.

==================================================
VERY IMPORTANT DECAL CHANGE RULE
==================================================

New validation decal required:
Anytime expiration date changes.

==================================================
JOINT OWNERSHIP RULE
==================================================

Plate in names joined:
By:
"AND" or "OR":
May transfer to:
Either owner.

==================================================
VERY IMPORTANT SPECIAL PLATE ELIGIBILITY RULE
==================================================

Special plate transfer:
Requires remaining owner:
Still eligible.

==================================================
VERY IMPORTANT PRORATION RULE
==================================================

Registration tax:
Prorated monthly:
By effective date.

==================================================
VERY IMPORTANT EFFECTIVE DATE RULE
==================================================

Effective date should reflect:
Date transfer should have occurred.

==================================================
VERY IMPORTANT TRANSFER OWNERSHIP SAME VEHICLE RULE
==================================================

If same vehicle remains:
Use:
Transfer Registration Ownership transaction.

==================================================
VERY IMPORTANT EXPIRED REGISTRATION RULE
==================================================

Expired registration:
Must be renewed or extended:
When transferred.

==================================================
ETR EXPIRATION RULE
==================================================

Electronic temporary transfer:
Expires:
When current registration expires.

==================================================
VERY IMPORTANT 30 DAY RULE
==================================================

ETR temporary transfer:
NOT allowed:
If less than 30 days remain:
On registration.

==================================================
VERY IMPORTANT PERMANENT TRANSFER RULE
==================================================

Permanent transfer:
Must occur:
Before renewal.

==================================================
VERY IMPORTANT LIMITED TRANSACTION RULE
==================================================

Until permanent transfer complete:
Only duplicate registration:
Allowed in system.

==================================================
VERY IMPORTANT DEAL FALL THROUGH RULE
==================================================

If deal falls through:
Dealer must:
Restore registration:
Back to original vehicle.
