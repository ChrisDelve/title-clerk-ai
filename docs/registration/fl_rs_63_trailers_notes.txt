==================================================
VERY IMPORTANT CORE RULE
==================================================

All trailers:
Must comply:
With Florida trailer registration requirements
based on:
- trailer type
- trailer weight
- trailer ownership
- trailer usage.

==================================================
TRAVEL TRAILER RULE
==================================================

Travel trailer:
Portable unit:
Designed for temporary living quarters:
Maximum:
- 8.5 feet body width
- 40 feet body length.

==================================================
VERY IMPORTANT LENGTH RULE
==================================================

Travel trailer fees:
Calculated:
Using trailer length:
INCLUDING hitch.

==================================================
CAMPING TRAILER RULE
==================================================

Camping trailer:
Portable unit:
With collapsible sidewalls:
Used for recreational/camping use.

==================================================
PARK TRAILER RULE
==================================================

Park trailer:
Transportable unit:
Maximum:
- 14 feet body width
- 400 or 500 square feet depending on standards.

==================================================
VERY IMPORTANT ANSI RULE
==================================================

Park trailers:
May be built:
Under:
- ANSI A-119.5 standards
OR
- HUD standards.

==================================================
FIFTH-WHEEL TRAILER RULE
==================================================

Fifth-wheel travel trailer:
Uses:
Tow mechanism:
Mounted above/forward of rear axle.

==================================================
VERY IMPORTANT FIFTH-WHEEL CLASS CODE RULE
==================================================

Use:
Class code 77 or 78:
For fifth-wheel trailers.

==================================================
TRAILER DEFINITION RULE
==================================================

Trailer:
Vehicle without motive power:
Drawn by motor vehicle:
No trailer weight rests on towing vehicle.

==================================================
VERY IMPORTANT BALL HITCH RULE
==================================================

Ball hitch trailers:
With living/storage space:
Use:
Class code 78.

==================================================
VERY IMPORTANT MOTORCYCLE PLATE RULE
==================================================

Motorcycle-size plates:
Permitted:
On trailers:
1,999 pounds or less.

==================================================
EQUIPMENT TRAILER RULE
==================================================

Equipment on wheels:
Without motive power:
Must register:
As trailer by weight.

==================================================
VERY IMPORTANT WEIGHT SLIP RULE
==================================================

If trailer weight unavailable:
Certified weight slip:
Required.

==================================================
SEMI-TRAILER RULE
==================================================

Semi-trailer:
Vehicle:
Part of weight/load:
Rests upon another vehicle.

==================================================
VERY IMPORTANT PERMANENT PLATE RULE
==================================================

Permanent semi-trailer plate:
Allowed:
If semi-trailer drawn:
By GVW truck tractor:
Using fifth-wheel arrangement.

==================================================
VERY IMPORTANT CLASS CODE 103 RULE
==================================================

Permanent semi-trailer plates:
Use:
Class code 103.

==================================================
VERY IMPORTANT NONTRANSFERABLE RULE
==================================================

Permanent semi-trailer plates:
- not renewed
- nontransferable
- only used on assigned trailer.

==================================================
VERY IMPORTANT NO ADVANCED REPLACEMENT RULE
==================================================

Permanent semi-trailer plates:
Not included:
In forced replacement program.

==================================================
OFFICE TRAILER RULE
==================================================

Office trailer:
Mobile office building:
May require registration:
When moved on roads.

==================================================
VERY IMPORTANT OFFICE TRAILER WEIGHT RULE
==================================================

Office trailer weight:
Calculated:
Length × width × 20 pounds.

==================================================
VERY IMPORTANT OFFICE TRAILER CLASS CODE RULE
==================================================

For-hire office trailers:
Use:
Class code 54.

==================================================
VERY IMPORTANT PRIVATE PROPERTY RULE
==================================================

Office trailers:
Parked on private property:
Do NOT require:
Valid registration.

==================================================
TOW DOLLY RULE
==================================================

Tow dolly:
Does NOT require:
Registration.

==================================================
VERY IMPORTANT TOWING VEHICLE RULE
==================================================

Tow dolly towing vehicle:
Must be:
Currently registered.

==================================================
CONVERTER GEAR RULE
==================================================

Converter gear:
Does NOT require:
Registration.

==================================================
VERY IMPORTANT DOUBLE RIG RULE
==================================================

Vehicles coupled to converter gear:
Must both:
Be registered.

==================================================
GENERAL INFORMATION RULES
==================================================

Slide-in campers:
Do NOT require:
Registration.

==================================================
VERY IMPORTANT NONRESIDENT RV RULE
==================================================

Nonresident recreational vehicles:
In Florida:
6 months or longer:
Must display:
Proof of registration.

==================================================
VERY IMPORTANT GOVERNMENT TRAILER RULE
==================================================

Government-owned trailers:
Must display:
Government license plate:
Even if not moved on roads.

==================================================
VERY IMPORTANT NONPROFIT TRAILER RULE
==================================================

Eligible nonprofit trailers:
May receive:
X-Series plate.

==================================================
VERY IMPORTANT CLASS CODE 53 RULE
==================================================

If nonprofit requests regular plate:
Use:
Class code 53.

==================================================
TITLE REQUIREMENT RULES
==================================================

If trailer required to be titled:
Must already be titled:
OR
Application for title submitted.

==================================================
VERY IMPORTANT UNDER-1999 RULE
==================================================

Trailers:
1,999 pounds or less:
- NOT required to be titled
- NOT required to be insured
- exempt from Initial Registration Fee.

==================================================
VERY IMPORTANT CHAIN OF OWNERSHIP RULE
==================================================

Untitled trailers:
Require:
Complete chain of ownership.

==================================================
NEW TRAILER DOCUMENTATION RULE
==================================================

New trailer:
Requires:
- MCO
OR
- bill of sale
showing complete ownership chain.

==================================================
VERY IMPORTANT 12-MONTH RETENTION RULE
==================================================

Copies of MCO/bill of sale:
Retained:
Minimum:
12 months.

==================================================
USED TRAILER DOCUMENTATION RULE
==================================================

Used trailer:
Requires:
Bill of sale:
Including:
- trailer description
- previous Florida plate number.

==================================================
VERY IMPORTANT NEVER REGISTERED RULE
==================================================

Used trailer:
Never registered in Florida:
Requires:
Certified weight slip:
Unless weight shown on documentation.

==================================================
VERY IMPORTANT TIN RULE
==================================================

If trailer has no VIN:
System generates:
TIN number.

==================================================
VERY IMPORTANT TIN RECOMMENDATION RULE
==================================================

TIN:
Recommended:
To be die-cut or affixed:
To trailer.

==================================================
VERY IMPORTANT REGISTRATION BILL OF SALE RULE
==================================================

Registration:
Prints blank bill of sale section:
For future ownership transfer.

==================================================
FEE RULES
==================================================

Fees:
Determined using:
HSMV 83140 License Plate Rate Chart.

==================================================
REPLACEMENT RULES
==================================================

Replacement requires:
- HSMV 83146
- applicable fees.

==================================================
VERY IMPORTANT STOLEN PLATE RULE
==================================================

If stolen plate/decal:
Reported to law enforcement:
Replacement fee waived.

==================================================
NONPROFIT MARINE TRAILER RULES
==================================================

Marine nonprofit trailer:
Exempt:
From most registration fees/taxes.

==================================================
VERY IMPORTANT NONPROFIT REQUIREMENT RULE
==================================================

Organization:
Must qualify:
Under:
501(c)(3).

==================================================
VERY IMPORTANT CLASS CODE 55 RULE
==================================================

Marine nonprofit trailer:
Uses:
Class code 55.

==================================================
VERY IMPORTANT REGISTRATION USE NP RULE
==================================================

Marine nonprofit trailer:
Uses:
Registration use NP.

==================================================
VERY IMPORTANT BUSINESS CUSTOMER RULE
==================================================

Registration use NP:
Restricted:
To business customers only.

==================================================
VERY IMPORTANT SPECIALTY PLATE RULE
==================================================

Nonprofit marine trailers:
May receive:
Specialty/personalized plates:
But extra fees apply.

==================================================
VERY IMPORTANT BIENNIAL RULE
==================================================

Biennial registration:
Allowed:
For registration use NP.

==================================================
PROOF OF ELIGIBILITY RULE
==================================================

Original/copy:
IRS Determination Letter:
Required.

==================================================
VERY IMPORTANT FEIN RULE
==================================================

FEIN on IRS letter:
Must match:
Motor vehicle issuance system customer record.

==================================================
VERY IMPORTANT CUSTOMER MAINTENANCE RULE
==================================================

FEIN corrections:
Made:
Using Customer Maintenance.

==================================================
VERY IMPORTANT VIRTUAL OFFICE RULE
==================================================

Marine nonprofit trailer renewals:
May renew:
- in person
- by mail
- through Virtual Office.

==================================================
EXHIBIT A RULE
==================================================

Exhibit A includes:
Frequently Asked Questions:
Regarding:
- non-titled trailer sales
- homemade trailers
- trailer dealer licensing.

==================================================
VERY IMPORTANT COMPLETE CHAIN EXAMPLE RULE
==================================================

Complete chain of ownership:
Requires:
All prior bills of sale:
Connecting ownership history.

==================================================
VERY IMPORTANT HOMEMADE TRAILER RULE
==================================================

Homemade/assembled trailers:
2,000 pounds or more:
Require:
Compliance Examiner inspection.

==================================================
VERY IMPORTANT FLT NUMBER RULE
==================================================

Compliance Examiner:
Assigns:
FLT number:
To homemade trailers.

==================================================
VERY IMPORTANT FLT MARKING RULE
==================================================

FLT number:
Must be:
- die-cut
OR
- welded permanently:
To trailer.

==================================================
VERY IMPORTANT TL-43 RULE
==================================================

Homemade trailer processing:
Governed additionally by:
TL-43.

==================================================
VERY IMPORTANT DEALER LICENSE RULE
==================================================

Trailer dealers:
Selling:
2,000 pounds or more:
Need dealer license:
If selling:
25 or more annually.
