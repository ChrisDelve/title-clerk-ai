==================================================
VERY IMPORTANT CORE RULE
==================================================

Holder of valid FCC amateur radio station license:
May receive:
Special license plate:
Displaying official amateur radio call letters.

==================================================
VERY IMPORTANT BOTTOM VERBIAGE RULE
==================================================

Plate must display:
"Amateur Radio"
Embossed:
On bottom of plate.

==================================================
VERY IMPORTANT COUNTY ISSUANCE RULE
==================================================

Amateur Radio plates:
Ordered and issued:
By county tax collector office.

==================================================
FEE RULES
==================================================

Fees include:
- Original plate fee
- Decal on Demand fee
- Amateur Radio initial fee
- Service fee
- Branch fee if applicable
- Registration tax

==================================================
VERY IMPORTANT FEE DETERMINATION RULE
==================================================

Registration tax determined by:
- License plate inventory code
- Registration use
- Vehicle type

==================================================
VERY IMPORTANT FEE CHART RULE
==================================================

Refer:
Registration Fees and Flat Taxes Distribution Chart:
For applicable fees.

==================================================
ELIGIBLE VEHICLE RULES
==================================================

Eligible vehicles:
- Private automobiles
- Leased automobiles
- Trucks 7,999 lbs or less
- Travel trailers
- Camping trailers
- Motor homes
- Truck campers
- Private motor coaches

==================================================
VERY IMPORTANT COMMERCIAL USE RULE
==================================================

RV-type units:
Cannot be:
For hire
OR
Commercial use.

==================================================
VERY IMPORTANT MOTORCYCLE RULE
==================================================

Amateur Radio plates:
Cannot issue:
On motorcycles.

==================================================
VERY IMPORTANT RS-27 RULE
==================================================

Motorcycle customers:
May process:
Amateur radio call letters:
As personalized motorcycle plate:
Under RS-27.

==================================================
RESIDENCY REQUIREMENTS
==================================================

Applicant must be:
Florida resident.

==================================================
FCC LICENSE REQUIREMENTS
==================================================

Require:
Copy of valid FCC amateur radio station license.

==================================================
VERY IMPORTANT FCC PAPER LICENSE RULE
==================================================

FCC no longer mails:
Paper amateur radio licenses.

==================================================
VERY IMPORTANT FCC SEARCH RULE
==================================================

Customer may retrieve:
Reference copy:
Through FCC Universal Licensing System.

==================================================
FCC WEBSITE PROCESS
==================================================

Customer may:
1. Visit FCC Universal Licensing System
2. Select "Search Licenses"
3. Search "By Call Sign"
4. Enter call sign
5. Select matching license record
6. Click blue call sign
7. Open "Reference Copy"
OR
8. Use "Printable Page"

==================================================
VERY IMPORTANT CALL SIGN CHANGE RULE
==================================================

If FCC issues:
New call sign:
Customer must:
Reapply:
As first-time applicant.

==================================================
VERY IMPORTANT CREDIT RULE
==================================================

When changing to new call sign:
Credit allowed:
On old plate:
Additional fees collected.

==================================================
VERY IMPORTANT LETTER O RULE
==================================================

If new call sign contains:
Letter "O":
Must enter:
Letter "O"
Correctly in issuance system.

==================================================
APPLICATION REQUIREMENTS
==================================================

Require:
HSMV 83041.

==================================================
VERY IMPORTANT FORM RULE
==================================================

HSMV 83041:
Must be:
Legibly completed in detail.

==================================================
INSURANCE REQUIREMENTS
==================================================

Florida insurance required.

==================================================
VERY IMPORTANT RS-36 RULE
==================================================

Insurance requirements:
Governed by:
RS-36.

==================================================
RENEWAL AND TRANSFER RULES
==================================================

Renewals and transfers:
May process:
Through any county license plate agency.

==================================================
REPLACEMENT RULES
==================================================

Replacement plates/decals:
May process:
Through any county license plate agency.

==================================================
REPLACEMENT REQUIREMENTS
==================================================

Require:
- HSMV 83146
- Applicable fees
- Current registration copy

==================================================
REPLACEMENT FEES RULE
==================================================

Replacement fees include:
- Replacement fee
- Decal on Demand fee
- Service fee
- Branch fee if applicable
- Air Pollution Control fee
- Reflectorization fee
- EMS fee
- Advanced Replacement fee
- FRVIS fee

==================================================
VERY IMPORTANT STOLEN PLATE RULE
==================================================

If plate/decal stolen:
Replacement:
No charge:
With police report.

==================================================
VERY IMPORTANT LAW ENFORCEMENT CARD RULE
==================================================

Law enforcement agency card:
Containing case number:
May substitute:
For police report.

==================================================
VERY IMPORTANT COMMENT RULE
==================================================

Issuance system comments:
Must include:
- Law enforcement agency name
- Stolen report case number

==================================================
VERY IMPORTANT SP CODE RULE
==================================================

Replacement using code:
"SP":
Requires:
Police report copy
OR
Law enforcement card attached:
To HSMV 83146.

==================================================
VERY IMPORTANT NO REPORT RULE
==================================================

If no police report provided:
Fees required:
With HSMV 83146.

==================================================
VERY IMPORTANT CORE CONCEPT
==================================================

Florida Amateur Radio license plates:
Allow qualifying FCC-licensed Florida residents
to display official amateur radio call signs
on eligible private-use vehicles,
subject to strict FCC verification,
vehicle restrictions,
and replacement documentation requirements.
