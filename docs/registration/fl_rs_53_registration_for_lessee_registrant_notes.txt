==================================================
VERY IMPORTANT CORE RULE
==================================================

Long-term leased motor vehicles:
Must be registered:
In name and address:
Of lessee:
If class code is registrant possible.

==================================================
DEFINITION — ASSUMPTION
==================================================

Assumption:
Means:
Transfer of license plate ownership:
From lessor/owner:
To lessee/registrant.

==================================================
DEFINITION — REGISTRANT
==================================================

Registrant:
Means:
Lessee required:
To register vehicle:
In their own name:
Instead of lessor/owner.

==================================================
DEFINITION — REGISTRANT POSSIBLE
==================================================

Registrant possible:
Means:
Registration may exist:
In different name:
Than titled owner.

==================================================
VERY IMPORTANT VEHICLE CLASS INQUIRY RULE
==================================================

To verify registrant possible:
Use:
Miscellaneous Inquiry
→ Misc. Registration Inquiries
→ Vehicle Class Combination.

==================================================
VERY IMPORTANT CHECKMARK RULE
==================================================

If class code:
Is registrant possible:
System automatically displays:
Checkmark:
In "registrant possible" box.

==================================================
VERY IMPORTANT TITLE RULE
==================================================

Valid Florida title:
Must exist:
Before lessee/registrant plate issuance.

==================================================
VERY IMPORTANT CLASS CODE RESTRICTION RULE
==================================================

If class code:
NOT registrant possible:
Lessee registration:
Cannot issue.

==================================================
FIRST-TIME FLORIDA TITLE RULE
==================================================

When lease vehicle:
First titled in Florida:
Registration issues:
In lessee/registrant name:
If class code qualifies.

==================================================
DOCUMENTATION RULES
==================================================

Customer must provide:
Documentation verifying:
- lessee name
AND
- vehicle description.

==================================================
ACCEPTABLE DOCUMENTATION RULE
==================================================

Acceptable documentation includes:
- lease agreement
- power of attorney
- lease documentation
showing:
- lessee name
- vehicle description.

==================================================
PLATE ASSUMPTION RULES
==================================================

Older plates:
Previously issued:
In lessor/owner name:
May be assumed:
By lessee/registrant.

==================================================
VERY IMPORTANT SAME VEHICLE RULE
==================================================

Assumed plate:
Must remain:
Assigned:
To same vehicle.

==================================================
VERY IMPORTANT ASSUMPTION DOCUMENT RULE
==================================================

Plate assumption:
Requires:
Lease documentation:
Verifying:
- lessee name
- vehicle description.

==================================================
PLATE TRANSFER RULES
==================================================

If lessee already has:
Plate issued in their name:
Plate may transfer:
To:
- another leased vehicle
OR
- privately owned vehicle.

==================================================
VERY IMPORTANT DECAL RULE
==================================================

If registration period extended:
New decal required:
Even if expiration year unchanged.

==================================================
RENTAL/LEASE DOCUMENTATION RULES
==================================================

Vehicle operator:
Must possess:
Registration certificate
OR
True copy of lease/rental documentation.

==================================================
VERY IMPORTANT REQUIRED LEASE INFORMATION RULE
==================================================

Lease/rental documentation must contain:
- rental date
- exit time
- rental station ID
- agreement number
- VIN
- plate number/state
- make/model/color
- mileage
- authorized renter name.

==================================================
SPECIAL PLATE ELIGIBILITY RULES
==================================================

Lessee/registrants:
May qualify:
For:
- specialty plates
- military plates
- disabled plates
- amateur radio plates
- fleet plates
- government plates
subject to qualification rules.

==================================================
VERY IMPORTANT GOVERNMENT PLATE RULE
==================================================

Government plates:
With agency acronym:
Processed ONLY:
Through Division of Motorist Services.

==================================================
VERY IMPORTANT MEDAL OF HONOR RULE
==================================================

Certain military plates:
Available ONLY:
Through Division of Motorist Services.

==================================================
FEE RULES
==================================================

Applicable fees include:
- registration tax
- trust fund fees
- service fee
- branch fee
- license plate fee
- specialty annual use fee
- Initial Registration Fee if applicable.

==================================================
VERY IMPORTANT IRF RULE
==================================================

Initial Registration Fee:
May apply:
When transferring:
From lease class code:
To class code subject to IRF.

==================================================
VERY IMPORTANT RS-30 RULE
==================================================

IRF exemptions:
Governed by:
RS-30.

==================================================
PROOF OF OWNERSHIP RULES
==================================================

Acceptable proof includes:
- Florida registration certificate
- Florida title copy
- title application receipt.

==================================================
INSURANCE RULES
==================================================

Florida insurance:
Required.

==================================================
VERY IMPORTANT RS-36 RULE
==================================================

Insurance processing:
Governed by:
RS-36.

==================================================
RENEWAL AND TRANSFER RULES
==================================================

Renewals/transfers:
May process:
At county tax collector or license plate agent.

==================================================
REPLACEMENT RULES
==================================================

Replacement requires:
- HSMV 83146
- registration certificate copy
- applicable fees.

==================================================
VERY IMPORTANT REPLACEMENT FEE RULE
==================================================

Replacement fees may include:
- replacement fee
- service fee
- branch fee
- Air Pollution fee
- EMS fee
- FRVIS fee
- Reflectorization fee
- Advanced Replacement fee
- Decal on Demand fee.

==================================================
PERSONALIZED PLATE RESERVATION RULE
==================================================

Personalized plate:
May reserve:
In lessee/registrant name:
If class code is registrant possible.

==================================================
VERY IMPORTANT SAME NAME RULE
==================================================

Reserved personalized plate:
Must issue:
In same name:
As reservation.

==================================================
INQUIRY RULES
==================================================

Title/VIN inquiry:
Displays:
Lessor/owner information.

==================================================
VERY IMPORTANT LICENSE PLATE INQUIRY RULE
==================================================

Plate/decal inquiry:
Displays:
Lessee/registrant information.

==================================================
VERY IMPORTANT TITLE DATA RULE
==================================================

Separate title data:
Shows:
Actual title owner.

==================================================
VERY IMPORTANT TAG HISTORY RULE
==================================================

Tag history:
Displays:
Registrant name:
At time of transaction.

6. Is plate assumption allowed?
7. Is assumed plate staying on same vehicle?
8. Is plate transfer allowed?
9. Is IRF applicable?
10. Does IRF exemption apply?
11. Is Florida insurance attached?
12. Is specialty/military plate qualification satisfied?
13. Is government plate restricted to DMS?

==================================================
VERY IMPORTANT CORE CONCEPT
==================================================

Florida lessee/registrant registration procedures:
Allow qualifying leased vehicles
to register in lessee names
instead of titled lessor names,
while enforcing:
- registrant possible class codes
- valid Florida title requirements
- lease documentation verification
- insurance compliance
- controlled plate assumptions/transfers
and specialty plate qualification rules.
