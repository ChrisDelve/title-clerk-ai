- Documentation requirements

==================================================
VERY IMPORTANT CORE RULE
==================================================

If gas-powered bicycle:
- does not exceed 2 brake horsepower
- does not exceed 30 MPH
- does not exceed 50cc

Vehicle:
Classifies as:
MOPED.

==================================================
VERY IMPORTANT MOPED RULE
==================================================

Qualifying gas-powered bicycle:
MUST be:
Registered:
BUT NOT titled.

==================================================
VERY IMPORTANT ELECTRIC BICYCLE RULE
==================================================

Electric helper motor bicycles:
Under statutory bicycle definition:
Are NOT registered or titled.

==================================================
VERY IMPORTANT MOTORCYCLE RULE
==================================================

If gas-powered bicycle:
- exceeds 2 brake horsepower
OR
- exceeds 30 MPH
OR
- exceeds 50cc

Vehicle:
Classifies as:
MOTORCYCLE.

==================================================
VERY IMPORTANT TL-43 RULE
==================================================

Motorcycle classification:
Requires:
Title and registration processing:
Under:
TL-43 assembled from parts procedures.

==================================================
VERY IMPORTANT DMS INSPECTION RULE
==================================================

Customer:
Must contact:
DMS Regional Office.

Compliance Examiner:
Inspects vehicle:
Before registration.

==================================================
VERY IMPORTANT FLA DECAL RULE
==================================================

If vehicle complies:
Compliance Examiner:
Assigns:
FLA VIN decal.

==================================================
VERY IMPORTANT MOTORCYCLE PLATE RULE
==================================================

Regional Office:
Issues:
Registration and motorcycle plate:
For qualifying moped.

==================================================
VERY IMPORTANT ASPT RULE
==================================================

Assembled gas-powered bicycle:
Uses:
MAKE:
"ASPT"

==================================================
VERY IMPORTANT MODEL YEAR RULE
==================================================

Model year:
Equals:
Year vehicle assembled.

==================================================
VERY IMPORTANT COMPLETED STATE RULE
==================================================

Vehicle:
Must be fully completed:
Before inspection/VIN assignment.

==================================================
VERY IMPORTANT ORIGINAL DOCUMENT RULE
==================================================

Customer:
Must provide:
Original documentation:
To Compliance Examiner.

==================================================
VERY IMPORTANT REJECTION RULE
==================================================

If inspection rejected:
Compliance Examiner:
Retains paperwork:
Until deficiencies corrected.

==================================================
INSPECTION FEE RULES
==================================================

Initial inspection fee:
$40.

Subsequent inspections:
$20 each.

==================================================
DOCUMENTATION REQUIREMENTS
==================================================

Required documents include:
- HSMV 84490
- original bills of sale
- original receipts
- affidavit for inspection
- sales tax documentation.

==================================================
VERY IMPORTANT HSMV 84490 RULE
==================================================

HSMV 84490:
Statement of Builder:
Must:
- be completed by customer
- include assigned FLA number
- be completed by Compliance Examiner.

==================================================
VERY IMPORTANT MAJOR COMPONENT RULE
==================================================

Major component parts:
Include:
- bicycle frame
- gasoline motor.

==================================================
VERY IMPORTANT ENGINE RECEIPT RULE
==================================================

Engine receipt:
Must include:
- engine year
- seller information
- purchaser information.

==================================================
VERY IMPORTANT CUSTOMER NAME RULE
==================================================

Bills of sale/receipts:
Must be:
In customer/builder name.

==================================================
VERY IMPORTANT AFFIDAVIT RULE
==================================================

Customer:
Must sign:
Affidavit for Inspection of a Motorized Bicycle Converted to a Moped.

==================================================
VERY IMPORTANT SALES TAX RULE
==================================================

Sales tax:
Collected:
On component parts:
If tax not previously paid.

==================================================
VERY IMPORTANT COMMENT FIELD RULE
==================================================

Regional Office:
Must enter:
- inspecting region number
- FLA decal audit number

Into:
Comment Desc. field.

==================================================
VERY IMPORTANT FLA DECAL PLACEMENT RULE
==================================================

FLA decal:
Must be affixed:
To bicycle frame.

==================================================
VERY IMPORTANT ODOMETER RULE
==================================================

Assembled from parts moped:
Odometer branded:
"NOT ACTUAL MILEAGE."

==================================================
VERY IMPORTANT DRIVER LICENSE RULE
==================================================

Driver license:
Required:
To operate gas-powered bicycle classified as moped.

==================================================
VERY IMPORTANT RULES OF ROAD RULE
==================================================

Moped operators:
Must comply with:
- rules of the road
- motorcycle safety equipment requirements.

==================================================
VERY IMPORTANT LAW ENFORCEMENT RULE
==================================================

Questions regarding:
Where vehicle may operate:
Should be directed:
To law enforcement.

VERY IMPORTANT CORE CONCEPT
==================================================

Florida gas-powered bicycle procedures:
Create a structured assembled-from-parts registration process
for bicycles converted into mopeds,
while enforcing:
- horsepower limitations
- speed limitations
- engine displacement limits
- Compliance Examiner inspections
- FLA VIN decal assignment
- documentation verification
and motorcycle reclassification thresholds.
