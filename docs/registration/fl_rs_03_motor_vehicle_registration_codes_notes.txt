==================================================
TAX CLASS CODE RULES
==================================================

Tax class code:
Determined by:
Combination of vehicle information.

==================================================
VERY IMPORTANT CLASS CODE RULE
==================================================

Class code created from:
- Vehicle weight
- Body type
- Vehicle type

==================================================
VERY IMPORTANT REGISTRATION USE RULE
==================================================

Registration use code:
Combined with:
- Inventory code
- Class code
To determine registration fees.

==================================================
VERY IMPORTANT CLASS CODE CONVERSION RULE
==================================================

Class code conversions:
Available through:
General Reports:
In motor vehicle issuance system.

==================================================
TAG TYPE CODE RULES
==================================================

Tag type identifier codes:
No longer used.

==================================================
VERY IMPORTANT INVENTORY CODE RULE
==================================================

License plate inventory codes:
Now used:
To determine registration fees.

==================================================
VERY IMPORTANT PERSONALIZED PLATE RULE
==================================================

Inventory code ending with:
"P"
Indicates:
Personalized plate.

==================================================
VERY IMPORTANT AMATEUR RADIO RULE
==================================================

Inventory code ending with:
"A"
Indicates:
Amateur radio plate.

==================================================
VERY IMPORTANT DEALER SPECIALTY RULE
==================================================

Inventory code ending with:
"J"
Indicates:
Dealer specialty plate.

==================================================
VERY IMPORTANT FLEET SPECIALTY RULE
==================================================

Inventory code ending with:
"O"
Indicates:
Fleet specialty plate.

==================================================
FEE CODE RULES
==================================================

Fee codes:
No longer used.

==================================================
VERY IMPORTANT REG USE FEE RULE
==================================================

Registration use codes:
Now determine:
- Charges
- Exemptions

==================================================
TRANSFER CODE RULES
==================================================

Transfer codes:
No longer used.

==================================================
VERY IMPORTANT TRANSFER CALCULATION RULE
==================================================

Motor vehicle issuance system:
Calculates:
Registration tax and fees:
Using:
- REG USE
- Vehicle weight

==================================================
VERY IMPORTANT TRANSFER FEE RULE
==================================================

Certain plate transfers:
Require:
$4.50 transfer fee.

==================================================
MAIL CODE RULES
==================================================

Mail code handling:
References:
RS-01.

==================================================
VERY IMPORTANT COUNTY MAIL FEE RULE
==================================================

Mail fees collected by county:
Retained by county.

==================================================
VERY IMPORTANT STATE MAIL FEE RULE
==================================================

Mail fees collected by FLHSMV:
Distributed to:
General Revenue Fund.

Registration taxes and fees.
