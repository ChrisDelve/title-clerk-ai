==================================================
VERY IMPORTANT CORE RULE
==================================================

Any tram:
Operated on Florida roads:
Must be registered.

==================================================
POWER UNIT RULES — NOT-FOR-HIRE
==================================================

Not-for-hire power units:
Registered:
As trucks:
Based on weight.

==================================================
VERY IMPORTANT GVW RULE
==================================================

If net weight:
Exceeds 5,000 pounds:
Gross Vehicle Weight:
Must be declared.

==================================================
VERY IMPORTANT PASSENGER WEIGHT RULE
==================================================

GVW calculation:
Includes:
- power unit net weight
PLUS
- passenger weight
In power unit only.

==================================================
VERY IMPORTANT TRAILING MODULE EXCLUSION RULE
==================================================

Do NOT include:
- trailing module weight
OR
- trailing module passengers
When calculating power unit GVW.

==================================================
POWER UNIT RULES — FOR-HIRE
==================================================

For-hire classifications:
Based on passenger count.

==================================================
VERY IMPORTANT CLASS CODE 31 RULE
==================================================

For-hire:
Up to 4 passengers:
Including driver:
Use:
Class code 31.

==================================================
VERY IMPORTANT CLASS CODE 09 RULE
==================================================

For-hire:
Over 4 passengers but less than 9:
Use:
Class code 09 (van).

==================================================
VERY IMPORTANT CLASS CODE 36 RULE
==================================================

For-hire:
Over 9 passengers:
Use:
Class code 36 (bus).

==================================================
TRAILING MODULE RULES
==================================================

Trailing modules:
Registered:
As trailers.

==================================================
VERY IMPORTANT TRAILER WEIGHT RULE
==================================================

If proof of ownership:
Does not show weight:
Weight may be calculated:
Width × Length × 20 pounds.

==================================================
VERY IMPORTANT CLASS CODE 53 RULE
==================================================

Not-for-hire trailing modules:
Use:
Class code 53.

==================================================
VERY IMPORTANT CLASS CODE 54 RULE
==================================================

For-hire trailing modules:
Use:
Class code 54.

==================================================
PROOF OF OWNERSHIP RULES
==================================================

Power unit:
Requires:
Manufacturer’s Certificate of Origin:
And title.

==================================================
VERY IMPORTANT TRAILING MODULE OWNERSHIP RULE
==================================================

Trailing module:
Acceptable proof:
- Manufacturer’s Certificate of Origin
OR
- bill of sale.

==================================================
VERY IMPORTANT 2,000 POUND TITLE RULE
==================================================

Trailing modules:
2,000 pounds or more:
Require title.

==================================================
INSURANCE RULES
==================================================

Proof of Florida insurance:
Required.

==================================================
VERY IMPORTANT RS-36 RULE
==================================================

Insurance processing:
Governed by:
RS-36.

==================================================
HVUT RULES
==================================================

If power unit GVW:
55,000 pounds or more:
HVUT proof required.

==================================================
VERY IMPORTANT RS-34 RULE
==================================================

HVUT processing:
Governed by:
RS-34.

==================================================
FEE RULES
==================================================

Required fees include:
- registration tax
- trust fund fees
- original plate fee
- Decal on Demand fee
- service fee
- branch fee if applicable.

==================================================
VERY IMPORTANT CLASSIFICATION RULE
==================================================

Fees:
Determined:
By:
- vehicle classification
- weight
- applicable class code.

==================================================
RENEWAL AND TRANSFER RULES
==================================================

Renewals and transfers:
May process:
At any county license plate agency.

==================================================
REPLACEMENT RULES
==================================================

Replacement requires:
- copy of registration certificate
- HSMV 83146
- applicable replacement fees.

==================================================
VERY IMPORTANT REPLACEMENT FEES RULE
==================================================

Replacement fees may include:
- replacement fee
- Decal on Demand fee
- service fee
- branch fee
- Air Pollution fee
- FRVIS fee
- Reflectorization fee
- EMS fee.

==================================================
VERY IMPORTANT STOLEN PLATE RULE
==================================================

If customer:
Provides police report or agency card:
Per RS-06:
Replacement fee waived.
