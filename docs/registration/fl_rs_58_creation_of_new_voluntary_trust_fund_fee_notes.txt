- Deauthorization standards

==================================================
VERY IMPORTANT CORE RULE
==================================================

New voluntary trust fund fees:
Require:
Legislative approval:
Before implementation.

==================================================
VERY IMPORTANT APPLICATION DEADLINE RULE
==================================================

Application requirements:
Must be submitted:
At least:
90 days before:
Next regular legislative session.

==================================================
VERY IMPORTANT RECOMMENDED DEADLINE RULE
==================================================

FLHSMV recommends:
Applications submitted:
No later than:
December 1.

==================================================
APPLICATION REQUIREMENTS
==================================================

Organization must submit:
- request letter
- application fee
- marketing strategy
- financial analysis
- Solicitation of Contributions proof if applicable.

==================================================
REQUEST LETTER REQUIREMENTS
==================================================

Request letter must include:
- organization name/address
- contact information
- proposed contribution amount
- recipient organization
- fiscal year end
- redistribution details
- intended use of funds
- renewal notice type requested.

==================================================
VERY IMPORTANT CONTRIBUTION TYPE RULE
==================================================

Organization must specify:
Whether contribution applies to:
- motor vehicle renewal notice
- driver license renewal notice
OR
both.

==================================================
APPLICATION FEE RULES
==================================================

Application fee:
$10,000:
Per requested renewal system.

==================================================
VERY IMPORTANT SEPARATE FEE RULE
==================================================

Motor vehicle renewal contribution:
$10,000 application fee.

Driver license renewal contribution:
Separate $10,000 application fee.

==================================================
VERY IMPORTANT PAYMENT RULE
==================================================

Payment must be payable to:
Division of Motorist Services.

==================================================
VERY IMPORTANT STATE FUNDS RULE
==================================================

State funds:
Cannot be used:
To pay application fee.

==================================================
VERY IMPORTANT REFUND RULE
==================================================

If Legislature:
Does not approve:
Requested contribution:
Application fee refunded.

==================================================
MARKETING STRATEGY RULE
==================================================

Organization must submit:
Short-term and long-term marketing plans.

==================================================
VERY IMPORTANT FINANCIAL ANALYSIS RULE
==================================================

Financial analysis:
Must identify:
- anticipated revenues
- planned expenditures.

==================================================
SOLICITATION OF CONTRIBUTIONS RULE
==================================================

Organization must provide:
Proof of compliance:
With Solicitation of Contributions Act:
If required under Chapter 496 FS.

==================================================
LEGISLATIVE PROCESS RULES
==================================================

If requirements satisfied:
FLHSMV notifies:
House and Senate Transportation Committees.

==================================================
VERY IMPORTANT SPONSOR RULE
==================================================

Organization:
Must obtain:
Legislative sponsors:
In BOTH:
- Senate
AND
- House.

==================================================
VERY IMPORTANT CONSISTENCY RULE
==================================================

Proposed legislation:
Must remain consistent:
With existing voluntary trust fund fees.

==================================================
IMPLEMENTATION RESPONSIBILITIES
==================================================

If approved:
Division of Motorist Services must:
- add fee to renewal systems
- develop computer programming
- distribute collected funds
- revise procedures.

==================================================
VERY IMPORTANT PROGRAMMING RULE
==================================================

FLHSMV responsible:
For computer programming:
Related to:
Collection and distribution.

==================================================
AUDIT RULES
==================================================

Organizations subject to:
215.97 FS:
Must submit:
Audit reports:
Per Auditor General rules.

==================================================
VERY IMPORTANT ANNUAL ATTESTATION RULE
==================================================

Organizations not subject to audit:
Must annually attest:
Funds used in compliance with law.

==================================================
VERY IMPORTANT 9-MONTH RULE
==================================================

Annual attestation:
Must be submitted:
Within:
9 months after fiscal year end.

==================================================
VERY IMPORTANT INTEREST REPORTING RULE
==================================================

Attestation/report:
Must include:
- interest earned
- expenditures made.

==================================================
DEAUTHORIZATION RULES
==================================================

FLHSMV must discontinue:
Voluntary contribution:
Under certain conditions.

==================================================
VERY IMPORTANT $25,000 RULE
==================================================

Contribution discontinued:
If less than:
$25,000:
Collected by end of:
5th year.

==================================================
VERY IMPORTANT SUBSEQUENT 5-YEAR RULE
==================================================

Contribution also discontinued:
If less than:
$25,000:
Collected during any later 5-year period.

==================================================
VERY IMPORTANT ORGANIZATION CLOSURE RULE
==================================================

Contribution discontinued:
If organization:
- no longer exists
OR
- stops authorized services.

==================================================
VERY IMPORTANT RECIPIENT REQUEST RULE
==================================================

Contribution may discontinue:
At organizational recipient request.

VERY IMPORTANT CORE CONCEPT
==================================================

Florida voluntary trust fund fee procedures:
Create structured legislative and compliance requirements
for organizations seeking voluntary contributions
through:
- motor vehicle renewal notices
- driver license renewal notices

while enforcing:
- legislative approval
- application fees
- audit/attestation compliance
- revenue accountability
- marketing and financial planning
- contribution performance thresholds
and organizational oversight standards.
