VERY IMPORTANT CORE RULE
==================================================

Wheelchair emblem motorcycle plates:
Provide same parking privileges:
As disabled parking permits:
Issued under:
320.0848 FS.

==================================================
ELIGIBILITY RULES
==================================================

Wheelchair emblem plates:
May issue:
To owner or lessee:
Of:
- motorcycle
- moped
- motorized disability access vehicle.

==================================================
VERY IMPORTANT FLORIDA RESIDENCY RULE
==================================================

Applicant:
Must reside:
In Florida.

==================================================
VERY IMPORTANT DISABILITY QUALIFICATION RULE
==================================================

Applicant:
Must qualify:
For disabled parking permit:
Under:
320.0848 FS.

==================================================
APPLICATION REQUIREMENTS
==================================================

Application requires:
HSMV 83007.

==================================================
VERY IMPORTANT PHYSICIAN CERTIFICATION RULE
==================================================

Physician’s Statement of Certification:
On HSMV 83007:
Must be completed:
By approved certifying authority.

==================================================
VERY IMPORTANT RS-38 RULE
==================================================

Additional acceptable disability proof:
Governed by:
RS-38.

==================================================
VERY IMPORTANT APPLICATION RETENTION RULE
==================================================

County license plate agency:
Must retain:
HSMV 83007 application.

==================================================
LESSEE RULES
==================================================

Wheelchair emblem plates:
May issue:
On leased vehicles.

==================================================
VERY IMPORTANT RS-53 RULE
==================================================

Lessee/registrant processing:
Governed by:
RS-53.

==================================================
FEE RULES
==================================================

Applicable fees include:
- registration tax
- trust fund fees
- original plate fee
- service fee
- branch fee
- Decal on Demand fee.

==================================================
VERY IMPORTANT CLASS CODE RULE
==================================================

Use:
Class code 65
OR
Class code 69:
As applicable.

==================================================
RENEWAL AND TRANSFER RULES
==================================================

Renewals/transfers:
May process:
Through any county tax collector
OR
license plate agency.

==================================================
REPLACEMENT RULES
==================================================

Replacement requires:
- copy of current registration certificate
- HSMV 83146
- applicable fees.

==================================================
VERY IMPORTANT REPLACEMENT FEES RULE
==================================================

Replacement fees may include:
- replacement fee
- service fee
- branch fee
- Air Pollution Control fee
- Reflectorization fee
- Emergency Medical Services fee
- FRVIS fee
- Advanced Replacement fee
- Decal on Demand fee.

==================================================
STOLEN PLATE RULES
==================================================

If plate/decal reported stolen:
Replacement:
May issue:
At no charge.

==================================================
VERY IMPORTANT POLICE REPORT RULE
==================================================

Customer must provide:
- police report
OR
- law enforcement agency card
Containing:
Case number.

==================================================
VERY IMPORTANT COMMENT RULE
==================================================

Motor vehicle issuance system comments:
Must include:
- law enforcement agency name
- stolen report case number.

==================================================
VERY IMPORTANT SP CODE RULE
==================================================

If replacement uses:
Code "SP":
Copy of police report or law enforcement card:
Must attach:
To HSMV 83146.

==================================================
VERY IMPORTANT NO REPORT RULE
==================================================

If no police report/card provided:
Replacement fees required.
