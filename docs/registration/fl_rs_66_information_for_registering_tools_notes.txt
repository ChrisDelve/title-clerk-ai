==================================================
VERY IMPORTANT CORE RULE
==================================================

A tool:
Is a street-legal special purpose motor vehicle:
Designed exclusively:
For:
- drilling
- excavation
- construction
- spraying
OR
similar activity

and NOT designed or used:
To transport loads over public roads.

==================================================
VERY IMPORTANT LOAD RULE
==================================================

Primary determination question:
"Do you haul a load with this vehicle
in addition to using it as a tool?"

==================================================
VERY IMPORTANT ANY LOAD RULE
==================================================

Hauling a load:
Means:
ANY load.

==================================================
VERY IMPORTANT HEAVY TRUCK RULE
==================================================

If vehicle:
- carries tools/supplies
OR
- tows trailer

DO NOT register as tool.

Register as:
Heavy truck:
Class code 41.

==================================================
VERY IMPORTANT TRAILER RULE
==================================================

Tools:
Can ONLY be:
Street-legal motorized vehicles.

NEVER trailers.

==================================================
VERY IMPORTANT OFF-ROAD RULE
==================================================

Do NOT title or register:
- frontend loaders
- bulldozers
- road graders
- tractors
OR
similar off-road equipment.

==================================================
VERY IMPORTANT STREET LEGAL RULE
==================================================

Equipment:
Must be legally drivable:
On streets/highways:
To qualify as tool.

==================================================
RESTRICTED PLATE RULES
==================================================

Tool registrations:
Receive:
Restricted license plate.

==================================================
VERY IMPORTANT RSR RULE
==================================================

Restricted tool plates:
Use inventory code:
RSR.

==================================================
VERY IMPORTANT RESTRICTED USE RULE
==================================================

RSR inventory code:
Indicates:
Restricted operational use.

==================================================
CLASS CODE RULES
==================================================

Tool registrations:
Use:
Class code 94.

==================================================
VERY IMPORTANT TO VEHICLE TYPE RULE
==================================================

Vehicle type:
"TO"
Represents:
Tool vehicle.

==================================================
TITLE RULES
==================================================

Vehicle types requiring title:
Must be titled:
As tool vehicle.

==================================================
INSURANCE RULES
==================================================

Vehicles registered:
With vehicle type TO:
Do NOT require:
Proof of insurance.

==================================================
YES — EXAMPLES OF TOOLS
==================================================

Examples of street-legal tools include:
- crane
- concrete pump truck
- excavator/ditch digger
- pump truck
- tree spade
- transportable speed sign
- stump grinder
- grill on wheels
when mounted on street-legal motor vehicle.

==================================================
NO — EXAMPLES OF TRAILERS
==================================================

The following trailer-mounted equipment:
Are NOT tools:
- trailer concrete pump
- trailer grill on wheels
- trailer transportable speed sign
- trailer tree spade
- trailer stump grinder.

==================================================
VERY IMPORTANT RS-63 RULE
==================================================

Trailer-mounted equipment:
Governed by:
RS-63.

==================================================
NO — CONVERTER GEAR RULE
==================================================

Converter gear:
Not titled or registered.

==================================================
VERY IMPORTANT DOUBLE RIG RULE
==================================================

Vehicles attached:
To converter gear:
Must both:
Be registered.

==================================================
NO — TOW DOLLY RULE
==================================================

Tow dolly:
Not titled or registered.

==================================================
VERY IMPORTANT TOWING VEHICLE RULE
==================================================

Towing vehicle:
Must maintain:
Current registration.

==================================================
NO — OFF-ROAD EQUIPMENT RULE
==================================================

Off-road/non-street-legal equipment:
Not titled or registered.

==================================================
EXAMPLES OF NON-REGISTERABLE EQUIPMENT
==================================================

Examples include:
- street sweeper
- tractor
- grader
- bulldozer
- frontend loader.

==================================================
VERY IMPORTANT CORE CONCEPT
==================================================

Florida tool registration procedures:
Provide special registration classifications
for street-legal special purpose vehicles
used exclusively for:
- drilling
- excavation
- construction
- spraying
and similar industrial activities,

while enforcing:
- no-load transportation restrictions
- restricted-use registration
- street-legal requirements
- trailer exclusions
- heavy truck distinctions
and off-road equipment exclusions.
