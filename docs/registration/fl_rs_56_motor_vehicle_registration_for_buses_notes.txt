==================================================
VERY IMPORTANT CORE RULE
==================================================

All buses:
Operating on Florida roads:
Must be registered:
Using proper class code and registration use.

==================================================
PRIVATE USE BUS RULES
==================================================

Private use buses:
May register:
Under class code 01.

==================================================
PRIVATE USE EXAMPLES
==================================================

Examples include:
- large family bus use
- migrant worker transportation
- employee parking shuttle transportation.

==================================================
VERY IMPORTANT IRF RULE
==================================================

Initial Registration Fee:
Required:
When registered:
Under class code 01.

==================================================
SCHOOL BUS RULES
==================================================

School buses:
Used exclusively:
For school/church transportation:
Within county:
Use:
Class code 92 (Restricted).

==================================================
FOR-HIRE BUS RULES
==================================================

For-hire buses:
8 passengers or less:
Including driver:
Use:
Class code 09.

==================================================
VERY IMPORTANT CLASS CODE 36 RULE
==================================================

For-hire buses:
9 passengers or more:
Use:
Class code 36.

==================================================
SPECIAL INTEREST BUS RULE
==================================================

Special interest buses:
Use:
Class code 37.

==================================================
VERY IMPORTANT IRP RULE
==================================================

Buses transporting passengers:
State-to-state:
Picking up and dropping off passengers:
Require:
IRP registration.

==================================================
VERY IMPORTANT CHARTER RULE
==================================================

Chartered interstate buses:
Must use:
International Registration Plan (IRP).

==================================================
WEIGHT RULES
==================================================

Registration fee:
Based on:
Net weight.

==================================================
VERY IMPORTANT GVW RULE
==================================================

Gross Vehicle Weight:
Must be entered:
To determine:
State Transportation fee.

==================================================
VERY IMPORTANT PASSENGER FORMULA RULE
==================================================

GVW calculation:
Passenger count × 150 pounds
PLUS
Net vehicle weight.

==================================================
AUTONOMOUS VEHICLE RULES
==================================================

Autonomous buses:
Processed:
As standard bus transactions:
Using proper class code.

==================================================
VERY IMPORTANT FEID RULE
==================================================

Autonomous fleet buses:
Should be titled and registered:
In company name:
Using FEID number.

==================================================
VERY IMPORTANT GOVERNMENT AV RULE
==================================================

Government entities:
Must process AV buses:
Like other government-owned vehicles.

==================================================
VERY IMPORTANT TL-68 RULE
==================================================

Autonomous vehicle processing:
Also governed by:
TL-68.

==================================================
AUTONOMOUS VEHICLE DOCUMENT REQUIREMENTS
==================================================

Required documentation:
- 82040 MV
- proof of insurance.

==================================================
VERY IMPORTANT AV INSURANCE RULE
==================================================

Autonomous vehicle insurance:
Must include:
- minimum $1 million liability coverage
- bodily injury coverage
- property damage coverage
- PIP coverage
- uninsured/underinsured coverage.

==================================================
ISSUANCE RULES
==================================================

County tax collector offices:
May issue:
All bus plates:
Except IRP buses.

==================================================
VERY IMPORTANT IRP ISSUANCE RULE
==================================================

IRP buses:
Cannot be issued:
Through standard county issuance process.

==================================================
REPLACEMENT RULES
==================================================

Replacement requires:
- HSMV 83146
- registration certificate copy.

==================================================
VERY IMPORTANT RS-06 RULE
==================================================

Replacement processing:
Governed by:
RS-06.
