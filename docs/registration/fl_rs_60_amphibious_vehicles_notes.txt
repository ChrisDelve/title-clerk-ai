==================================================
VERY IMPORTANT CORE RULE
==================================================

Amphibious vehicles:
Must be registered:
As BOTH:
- motor vehicles
AND
- vessels.

==================================================
VERY IMPORTANT MOTOR VEHICLE RULE
==================================================

Amphibious vehicles:
Are included:
Under definition of:
Motor vehicle.

==================================================
VERY IMPORTANT TITLE RULE
==================================================

Florida Certificate of Title:
Required:
For amphibious vehicles.

==================================================
VERY IMPORTANT NO VESSEL TITLE RULE
==================================================

If motor vehicle title issued:
No separate vessel title:
Issued under vessel statutes.

==================================================
VESSEL REGISTRATION RULE
==================================================

Even if motor vehicle title exists:
Amphibious vehicle:
Still requires:
Vessel registration.

==================================================
VERY IMPORTANT WATER OPERATION RULE
==================================================

All vessels operated:
On Florida waters:
Must be registered:
Unless statutory exemption applies.

==================================================
VESSEL EXEMPTIONS
==================================================

Exempt vessels include:
- vessels operated exclusively on private lakes/ponds
- U.S. Government vessels
- ship lifeboats
- qualifying non-motor-powered vessels.

==================================================
MOTOR VEHICLE PROCESSING RULES
==================================================

Motor vehicle title application:
Processed:
Using standard motor vehicle requirements.

==================================================
VERY IMPORTANT AU VEHICLE TYPE RULE
==================================================

Motor vehicle processing:
Uses:
Vehicle type:
"AU".

==================================================
INSURANCE RULES
==================================================

Proof of Florida insurance:
Required.

==================================================
VERY IMPORTANT CLASSIFICATION RULE
==================================================

Amphibious vehicles:
May classify as:
- passenger vehicle
- bus
- for-hire vehicle

depending on title documentation.

==================================================
VERY IMPORTANT FEE RULE
==================================================

Motor vehicle registration fees:
Based on:
- class code
- vehicle weight.

==================================================
VESSEL PROCESSING RULES
==================================================

Vessel registration:
Requires:
- copy of HSMV 82041
- HSMV 87244.

==================================================
VERY IMPORTANT REGISTRATION ONLY RULE
==================================================

If motor vehicle title already issued:
Vessel processing:
Creates:
Registration-only vessel record.

==================================================
VERY IMPORTANT DUPLICATE VIN RULE
==================================================

Vessel registration:
Creates:
Duplicate VIN record:
Using vessel vehicle type.

==================================================
VERY IMPORTANT VS VEHICLE TYPE RULE
==================================================

Vessel processing:
Uses:
Vehicle type:
"VS".

==================================================
VERY IMPORTANT FL NUMBER RULE
==================================================

Upon vessel registration approval:
FL number:
And annual vessel decal:
Issued.

==================================================
VERY IMPORTANT VESSEL FEE RULE
==================================================

Vessel registration fees:
Based on:
Length of amphibious vehicle.

==================================================
FORMS REQUIRED

Commonly required forms:
- HSMV 82041
- HSMV 87244.
