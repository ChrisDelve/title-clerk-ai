SOURCE SUMMARY
============================================================

FLHSMV issued Information Notice INFO 24-023 on 09/05/2024.

Subject:
New Rule 15C-1.015, FAC — Vehicle Registration Information.

The notice explains that Rule 15C-1.015 defines a "valid passport" for vehicle registration applications under section 320.02(2)(a), Florida Statutes.

Section 320.02(2)(a), Florida Statutes, requires an individual applicant filing for vehicle registration to provide one of the following:

- Valid Florida driver license
- Valid Florida identification card
- Valid driver license or identification card issued by another state
- Valid passport

Rule 15C-1.015 defines what qualifies as a valid passport for registration purposes.

IMPORTANT:
This rule applies to vehicle registration applications.
This rule does NOT apply to title applications.

CORE RULE
============================================================

For an individual vehicle registration application under section 320.02(2)(a), Florida Statutes, the customer must provide one of the following:

- Valid driver license or identification card issued by Florida
- Valid driver license or identification card issued by another state
- Valid passport

Rule 15C-1.015 defines what "valid passport" means for this purpose.

VALID FOREIGN PASSPORT RULE
============================================================

A foreign passport must be unexpired AND must include or be accompanied by required lawful-presence documentation.

AND foreign_passport_is_unexpired = TRUE

THEN foreign_passport_may_satisfy_valid_passport_requirement_for_registration = TRUE

IF customer_provides_foreign_passport = TRUE
AND lawful_presence_documentation_missing = TRUE
THEN valid_passport_requirement_not_met_for_registration = TRUE

IF customer_provides_foreign_passport = TRUE
AND foreign_passport_expired = TRUE
THEN valid_passport_requirement_not_met_for_registration = TRUE

COMMON CLERK ANSWER:
A foreign passport by itself is not enough for Florida vehicle registration. The foreign passport must be unexpired and must include a DHS stamp/mark showing lawful presence, or be accompanied by an unexpired I-94, current permanent resident card, or unexpired immigrant visa.

DEALER SUBMISSION RULE
============================================================

Dealers submitting applications on behalf of an individual to register a vehicle must provide a copy of either:

1. A valid driver license or identification card issued by Florida or another state; OR
2. The valid passport information required by Rule 15C-1.015.

============================================================
TITLE-ONLY DISTINCTION
============================================================

Rule 15C-1.015 applies to vehicle registration applications.

The FLHSMV notice states this rule does NOT apply to title applications.

COMMON CLERK ANSWER:
This passport rule applies to registration applications, not title-only applications. If the customer is applying for registration or plate issuance, apply Rule 15C-1.015. If the transaction is truly title-only with no registration or plate issuance, this specific passport rule does not apply.

============================================================
REGISTRATION WORKFLOW
============================================================

1. Identify transaction type.
   - Title-only
   - Title with registration
   - Registration only
   - New plate
   - Transfer plate

2. If registration or plate issuance is involved, verify identification requirement.
   - Florida DL/ID
   - Out-of-state DL/ID
   - Valid passport

3. If U.S. passport:
   - Confirm it is unexpired.
   - U.S. passport card is also acceptable if unexpired.

4. If foreign passport:
   - Confirm it is unexpired.
   - Confirm lawful-presence documentation exists:
     - DHS stamp/mark
     - unexpired I-94
     - current permanent resident card
     - unexpired immigrant visa

5. If documentation is missing:
   - Hold registration application.
   - Advise customer to provide required documentation or obtain Florida DL/ID.

6. If title-only:
   - Do not apply Rule 15C-1.015 as a registration requirement.
   - Still follow all other title identification/signature requirements.

============================================================
COMMON DEALERSHIP QUESTIONS
============================================================

QUESTION:
Customer only has a U.S. passport. Can we register?

ANSWER:
Yes, an unexpired U.S. passport or U.S. passport card may satisfy the valid passport requirement for vehicle registration.

------------------------------------------------------------

QUESTION:
Customer has a foreign passport. Can we register?

ANSWER:
Only if the foreign passport is unexpired and includes required lawful-presence documentation, such as a DHS stamp/mark, unexpired I-94, current permanent resident card, or unexpired immigrant visa.

------------------------------------------------------------

QUESTION:
Customer has a foreign passport but no I-94, green card, immigrant visa, or DHS stamp. Can we register?

ANSWER:
Do not submit the registration application as-is. A foreign passport alone is not enough. Hold the registration application and request required lawful-presence documentation or have the customer obtain Florida DL/ID.

------------------------------------------------------------

QUESTION:
Does this passport rule apply to title-only?

ANSWER:
ANSWER:
No. INFO 24-023 states this new rule does not apply to title applications. It applies to vehicle registration applications. Title-only may proceed only through dealership title-only workflow / controller approval.

------------------------------------------------------------

QUESTION:
Customer has passport but no Florida ID. Can we issue a plate?

ANSWER:
Possibly. If it is an unexpired U.S. passport/passport card, it may satisfy the passport requirement. If it is a foreign passport, it must be unexpired and supported by required lawful-presence documentation. If registration/plate issuance is involved, do not proceed unless the required identification/passport documentation is present.
