============================================================

STATUTE:
319.001(11)

CONTROLS:
- Meaning of satisfaction of lien
- Full payment of debt
- Release of debtor by lienholder

AI ROUTING:
IF the user asks what counts as lien satisfaction or says a lien is paid, check 319.001(11) and TL-33.

OPERATIONAL RULES:

IF lien_paid_in_full = TRUE
THEN satisfaction_of_lien_review_required = TRUE

IF debtor_released_from_lien_by_lienholder = TRUE
THEN satisfaction_of_lien_exists = TRUE

IF customer_claims_paid_off = TRUE
AND proof_of_full_payment_missing = TRUE
AND lienholder_release_missing = TRUE
THEN satisfaction_of_lien_not_proven = TRUE

IF satisfaction_of_lien_not_proven = TRUE
THEN lien_removal_or_clean_title_processing_hold_required = TRUE

- Customer says lien is paid off.

- Can we remove lien if customer says paid?

QUESTION:
"Can one AND owner add a lien?"

ROUTE TO:
- 319.235_Operational_Notes.txt
- ownership_signature_logic_map.txt
- lien_logic_map.txt

EXPECTED ANSWER:
If ownership is AND, each co-owner must sign to place lien/encumbrance. If OR, one co-owner may generally execute the lien.

------------------------------------------------------------

QUESTION:
"Out-of-state title shows a lien."

ROUTE TO:
- 319.27_Operational_Notes.txt
- 319.23_Operational_Notes.txt
- lien_logic_map.txt

EXPECTED ANSWER:
Review out-of-state lien. When Florida title is first issued, liens shown on application, filed with department, or shown on existing out-of-state title must be reviewed/noted.

============================================================
REJECTION PREVENTION SUMMARY
============================================================

High rejection risk exists when:
- active lien is not resolved
- lien satisfaction is missing
- payoff proof is treated as automatic lien satisfaction without release review
- certified mail proof is missing for old lien removal
- 20-day certified mail timing is not satisfied
- 10-day protest period is ignored
- lienholder protests lien removal
- ELT status is ignored
- active ELT lien is treated as paper lien
- HSMV 82260 is submitted for active ELT lien
- lienholder name/address is missing
- vehicle/VIN information is missing on notice of lien
- out-of-state lien is ignored
- lien assignment documentation is missing
- AND co-owner lien lacks all required signatures
- lien priority/order is not reviewed
