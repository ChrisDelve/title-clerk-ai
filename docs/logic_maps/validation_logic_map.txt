VALID PASSPORT REGISTRATION IDENTIFICATION VALIDATION
============================================================

IF user_question_mentions = "passport"
OR user_question_mentions = "foreign passport"
OR user_question_mentions = "customer only has passport"
OR user_question_mentions = "passport but no Florida ID"
OR user_question_mentions = "valid passport for registration"
THEN validate_passport_registration_identification = TRUE

IF validate_passport_registration_identification = TRUE
THEN identify_transaction_type = TRUE

IF transaction_type_involves_registration_or_plate_issuance = TRUE
THEN Rule_15C_1_015_valid_passport_check_required = TRUE

IF transaction_type_is_title_only = TRUE
AND no_registration_or_plate_issuance_requested = TRUE
THEN Rule_15C_1_015_valid_passport_check_not_required_for_title_only = TRUE

IF customer_uses_US_passport = TRUE
THEN verify_US_passport_unexpired = TRUE

IF customer_uses_US_passport_card = TRUE
THEN verify_US_passport_card_unexpired = TRUE

IF customer_uses_foreign_passport = TRUE
THEN verify_foreign_passport_unexpired = TRUE

IF customer_uses_foreign_passport = TRUE
THEN verify_lawful_presence_documentation = TRUE

IF customer_uses_foreign_passport = TRUE
AND no_DHS_stamp_or_mark = TRUE
AND no_unexpired_I_94 = TRUE
AND no_current_permanent_resident_card = TRUE
AND no_unexpired_immigrant_visa = TRUE
THEN validation_failed_registration_ID = TRUE

IF passport_or_supporting_lawful_presence_documentation_expired = TRUE
THEN validation_failed_registration_ID = TRUE

IF validation_failed_registration_ID = TRUE
THEN hold_registration_application = TRUE

IF validation_failed_registration_ID = TRUE
THEN request_valid_DL_ID_or_required_passport_lawful_presence_documentation = TRUE

COMMON VALIDATION ANSWER:
For registration or plate issuance, validate whether the customer has a Florida/out-of-state DL or ID, or a valid passport. For a U.S. passport/passport card, confirm it is unexpired. For a foreign passport, confirm it is unexpired and supported by lawful-presence documentation: DHS stamp/mark, unexpired I-94, current permanent resident card, or unexpired immigrant visa. If those documents are missing or expired, hold registration. If the transaction is title-only with no registration, Rule 15C-1.015 does not apply.

============================================================
TITLE-ONLY DEALER EXCEPTION VALIDATION — SUPERVISOR WORKFLOW
============================================================

IF user_question_mentions = "title-only"
OR user_question_mentions = "title only"
OR user_question_mentions = "does not want to register"
OR user_question_mentions = "no registration"
THEN validate_title_only_exception = TRUE

IF validate_title_only_exception = TRUE
THEN determine_if_dealer_sale_transaction = TRUE

IF dealer_sale_transaction = TRUE
THEN default_workflow_should_be_title_and_registration = TRUE

IF validate_title_only_exception = TRUE
THEN verify_reason_for_title_only = TRUE

IF reason_for_title_only = "customer unresponsive"
THEN title_only_exception_reason_valid_for_review = TRUE

IF reason_for_title_only = "customer cannot meet state requirements"
THEN title_only_exception_reason_valid_for_review = TRUE

IF reason_for_title_only = "customer simply does not want to register"
THEN title_only_reason_not_sufficient_without_management_exception = TRUE

IF validate_title_only_exception = TRUE
THEN controller_approval_required = TRUE

IF financed_or_lienholder_involved = TRUE
THEN bank_approval_scanned_in_required_if_applicable = TRUE

IF validate_title_only_exception = TRUE
THEN intent_to_title_only_letter_required = TRUE

IF intent_to_title_only_letter_sent = TRUE
THEN ten_day_customer_response_period_required = TRUE

IF validate_title_only_exception = TRUE
THEN verify_no_plate_registration_or_transfer_tag_issued = TRUE

IF title_only_in_customer_state_possible = TRUE
THEN prefer_customer_state_title_only_review = TRUE

IF Florida_title_only_used = TRUE
THEN Florida_tax_due_review_required = TRUE

IF Florida_tax_paid = TRUE
THEN DR_123_available_to_show_taxes_paid_to_Florida = TRUE

IF financed_title_only = TRUE
THEN customer_may_need_state_to_state_title_transfer = TRUE

IF controller_approval_missing = TRUE
THEN validation_failed_hold_before_submission = TRUE

IF intent_to_title_only_letter_missing = TRUE
THEN validation_failed_hold_before_submission = TRUE

COMMON VALIDATION ANSWER:
For a dealer sale, title-only must be validated as an exception workflow. The normal process is title and registration. Acceptable reasons are normally customer unresponsive or customer cannot meet their state requirements. Confirm controller approval, bank approval if financed/applicable, and that the Intent to Title Only letter was sent with a 10-day response period. Confirm no plate, registration, or transfer tag will be issued. If title-only is processed in Florida, review Florida taxes due and DR-123. If financed, warn that state-to-state title transfer may be needed later.
