Applies when:
- Law enforcement agency controls vehicle/mobile home
- Property abandoned/lost/seized
- Title transfer required

==================================================
CORE LEGAL CONCEPTS
==================================================

LOST PROPERTY:
Property with:
- No identifiable owner
- Apparent value
- Mislaid on public property
- Substantially operable/functional

ABANDONED PROPERTY:
Property with:
- No identifiable owner
- Wrecked/inoperative/dismantled
OR
- No apparent intrinsic value

UNCLAIMED EVIDENCE:
Property:
- Seized by law enforcement
- Used in criminal/quasi-criminal case
- Unclaimed for 60 days after case disposition

==================================================
LAW ENFORCEMENT OPTIONS
==================================================

Agency may:
- Retain vehicle
- Transfer vehicle
- Donate vehicle
- Trade vehicle
- Sell vehicle
- Destroy vehicle (if low value)

==================================================
VERY IMPORTANT ABANDONED PROPERTY RULE
==================================================

If property abandoned on public property:
Officer:
- Places notice on vehicle
- Mails notice to owner/lienholder

==================================================
VERY IMPORTANT 5-DAY RULE
==================================================

If owner does not remove property:
- Within 5 days after posting/mailing
Agency may proceed with disposition.

==================================================
LOST PROPERTY HOLDING RULE
==================================================

Lost property:
- Must remain in custody 90 days

Agency:
- Publishes notice during first 45 days

==================================================
UNCLAIMED EVIDENCE RULE
==================================================

Unclaimed evidence:
- Must remain after case disposition
- Minimum 60 days

==================================================
GENERAL DOCUMENT REQUIREMENTS
==================================================

Most transactions require:
- HSMV 82040 MV
OR
- HSMV 82040 MH
- Notice documentation
- Certified mail proof
- Bill of sale
- Lien satisfactions
- Sales tax info
- Plate/decal info
- Title fees

==================================================
VERY IMPORTANT NOTICE REQUIREMENTS
==================================================

Notice must:
- Be physically placed on vehicle
- Be mailed certified mail
- Return receipt requested

Recipients:
- Owner
- Lienholder (if applicable)

==================================================
CERTIFIED MAIL PROOF REQUIREMENTS
==================================================

Need:
- Original return receipt
OR
- Certified copy
OR
- Returned unopened certified letter

==================================================
NOTICE CONTENT REQUIREMENTS
==================================================

Notice should include:
- Property description
- Public property location
- 5-day removal warning
- Disposal warning
- Removal/storage/publication cost liability
- Posting date
- Officer information
- Officer contact information

==================================================
VERY IMPORTANT PUBLIC SALE RULES
==================================================

If agency sells at public sale:
Need:
- Newspaper advertisement
- Bill of sale
- Lien satisfactions
- Sales tax handling
- Plate/decal handling
- Title fees

==================================================
PUBLICATION REQUIREMENTS
==================================================

Advertisement must:
- Publish once weekly
- 2 consecutive weeks
- Newspaper of general circulation

==================================================
NO NEWSPAPER RULE
==================================================

If no newspaper available:
Post advertisement:
- Courthouse door
- 3 public places
- At least 10 days before sale

==================================================
ADVERTISEMENT CONTENT REQUIREMENTS
==================================================

Advertisement must include:
- Make
- Year
- VIN
- Sale date
- Sale time
- Sale location
- Statement sale subject to liens

==================================================
SALE LOCATION RULE
==================================================

Sale held:
- Nearest suitable place
where property stored/held

==================================================
BILL OF SALE REQUIREMENTS
==================================================

Bill of sale must:
- Be on agency letterhead
- Include complete description
- Include make/year/VIN
- State sale subject to liens

==================================================
OUT-OF-STATE TITLE REQUIREMENTS
==================================================

Need:
- HSMV 82042
OR
- VIN verification on HSMV form

==================================================
VIN VERIFICATION EXEMPTIONS
==================================================

Verification NOT required:
- Mobile homes
- Trailers under 2,000 lbs
- Travel trailers
- Camping trailers
- Truck campers
- Fifth-wheel recreational trailers

==================================================
OUT-OF-STATE OWNERSHIP VERIFICATION
==================================================

Need verification from prior state:
- Ownership
- Lien status
- Vehicle description

==================================================
ACCEPTABLE OWNERSHIP VERIFICATION
==================================================

Acceptable:
- State printout
- Official state letter
- Copy of title
- Teletype printout

==================================================
UNKNOWN PREVIOUS STATE RULE
==================================================

If previous state unknown:
Need:
- Affidavit from agency

==================================================
ABANDONED PROPERTY AFFIDAVIT RULE
==================================================

For abandoned property:
Need notarized/perjury affidavit stating:
- Property abandoned
- Full description
- Intended disposition

==================================================
UNCLAIMED EVIDENCE AFFIDAVIT RULE
==================================================

Need affidavit stating:
- Property unclaimed evidence
- Proceedings complete 60+ days
- Intended disposition

==================================================
AGENCY RETENTION RULE
==================================================

If agency retains vehicle:
Sales tax area should state:
- "Retaining pursuant to s. 705.103"
OR
- "Retaining pursuant to s. 705.105"

depending on case type.

==================================================
LIEN SATISFACTION RULE
==================================================

Need:
- Satisfaction of all liens
shown in Florida or other states

==================================================
PLATE / DECAL RULES
==================================================

Need:
- License plate transfer/purchase
OR
- Non-use affidavit

Mobile homes:
- Non-use affidavit NOT allowed

==================================================
VERY IMPORTANT OPERATION OF LAW RULE
==================================================

If dealer/individual purchases:
Title must issue:
- In purchaser name first
before resale.

==================================================
LOW VALUE PROPERTY RULE
==================================================

If property not valuable:
Agency may:
- Destroy property
without titling first.

==================================================
REFUSE REMOVAL RULE
==================================================

If abandoned property low value:
Agency may:
- Notify refuse removal service
without titling first.

DEFINITION LOGIC
==================================================

LAW ENFORCEMENT OFFICER:
Certified officer authorized to:
- Bear arms
- Make arrests
- Enforce law

PUBLIC PROPERTY:
Government-owned:
- Roads
- Parks
- Sidewalks
- Rights-of-way
- Public land
