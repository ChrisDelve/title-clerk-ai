==================================================
DEFINITION
==================================================

An administrative stop:
- Is a computer entry on a title record
- Prevents title processing
- Is based on a court order
- Remains active until removed

==================================================
PURPOSE OF ADMINISTRATIVE STOP
==================================================

Administrative stops are used to:
- Prevent improper title transfers
- Freeze disputed ownership
- Protect court interests
- Prevent fraudulent activity
- Block title issuance
- Restrict registration/title transactions

==================================================
IMPORTANT SYSTEM EFFECT
==================================================

Once an administrative stop exists:
- Further title action is blocked
- Title transactions cannot proceed
- Ownership changes may be denied
- Processing remains suspended

UNTIL:
- Court authorizes removal

==================================================
REQUIRED DOCUMENTATION
==================================================

To place or remove a stop:

REQUIRED:
- Court order

Court order MUST:
- Be signed by a Judge
- Identify vehicle
- State reason for stop

==================================================
VEHICLE IDENTIFICATION REQUIREMENTS
==================================================

Court order must contain:
- Vehicle title number
OR
- VIN

==================================================
REQUESTER INFORMATION REQUIRED
==================================================

Request must include:
- Name
- Mailing address
- Email address
- Telephone number

==================================================
FLHSMV PROCESSING TIME
==================================================

Once received:
- FLHSMV places stop within 48 hours

==================================================
ACKNOWLEDGEMENT LETTER
==================================================

FLHSMV sends:
- Acknowledgement letter

Letter confirms:
- Stop placement
- Stop remains active
- Court controls release

==================================================
STOP REMOVAL RULE
==================================================

Administrative stop remains until:
- Subsequent court direction
OR
- Final court order

==================================================
VERY IMPORTANT RULE
==================================================

Administrative stops:
- REQUIRE court order

NOT acceptable:
- Verbal request
- Dealer request alone
- Customer request alone
- Attorney request without court order

==================================================
AI DECISION LOGIC
==================================================

AI should determine:

1. Is there an administrative stop?
2. Is there a signed court order?
3. Is stop placement requested?
4. Is stop removal requested?
5. Is VIN/title number included?
6. Is requester information complete?
7. Has court authorized removal?

==================================================
COMMON DEALERSHIP/TITLE ERRORS
==================================================

- Attempting stop without court order
- Missing judge signature
- Missing VIN/title number
- Assuming stop automatically expires
- Trying to process title with active stop
- Missing requester information
- Sending incomplete court documents

==================================================
IMPORTANT OPERATIONAL NOTES
==================================================

- Administrative stops are court-controlled
- FLHSMV cannot arbitrarily remove stops
- Court documentation is mandatory
- Stops freeze title processing
- Proper identification information required
- 48-hour placement window
- Stop remains until court authorizes release
