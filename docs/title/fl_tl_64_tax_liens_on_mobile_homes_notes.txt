LEGAL AUTHORITY
==================================================

Governed by:
- 320.19 FS

==================================================
VERY IMPORTANT TAX LIEN RULE
==================================================

Unpaid license tax:
Creates:
First lien:
Against mobile home.

==================================================
VERY IMPORTANT SUPERIOR LIEN RULE
==================================================

Tax lien:
Is superior:
To all other liens.

==================================================
VERY IMPORTANT 30-DAY RULE
==================================================

If taxes unpaid:
More than:
30 days:
Department may enforce lien.

==================================================
VERY IMPORTANT TAX WARRANT RULE
==================================================

Department may issue:
Tax warrant:
For unpaid taxes.

==================================================
DESCRIPTION AND USE
==================================================

Procedure explains:
- How to place tax liens
- Required notices
- Required forms
- Existing lienholder handling
- Title correction processing

==================================================
GENERAL INFORMATION
==================================================

Tax liens apply ONLY to:
Mobile homes requiring:
Mobile home decal.

==================================================
VERY IMPORTANT REAL PROPERTY RULE
==================================================

Tax liens do NOT apply to:
- Real property decals
- Mobile homes taxed as tangible personal property

==================================================
VERY IMPORTANT CUSTOMER NOTICE RULE
==================================================

Owner must be notified:
Before lien filed.

==================================================
VERY IMPORTANT WARNING RULE
==================================================

Agency must advise:
Lien may be placed:
If unpaid fees remain unpaid.

==================================================
VERY IMPORTANT RECORD RETENTION RULE
==================================================

Retain:
- Citations
- Letters
- Notifications

Related to lien case.

==================================================
FILING TAX LIEN — PRIMARY LIEN EXISTS
==================================================

When primary lien already exists:
Special notice procedures required.

==================================================
REQUIRED DOCUMENTS — EXISTING LIEN CASE
==================================================

Required:
1. Copy of notice letter to owner
2. Completed HSMV 82139
3. Certified mail notice to first lienholder
4. Returned certified receipt
5. Title correction processing
6. Original HSMV 82139
7. Certified receipt card

==================================================
VERY IMPORTANT OWNER LETTER RULE
==================================================

Agency must send:
Written notice:
Of fees due
AND
Proposed lien placement.

==================================================
VERY IMPORTANT HSMV 82139 RULE
==================================================

HSMV 82139:
Must be completed:
Sections 1, 2, and 4.

==================================================
VERY IMPORTANT LIENHOLDER NOTICE RULE
==================================================

Copy of HSMV 82139:
Must be sent:
Certified mail:
To first lienholder.

==================================================
VERY IMPORTANT CERTIFIED MAIL RULE
==================================================

Certified mail requires:
Return receipt requested.

==================================================
VERY IMPORTANT TITLE CORRECTION RULE
==================================================

Tax lien processed as:
- Title Correction

==================================================
VERY IMPORTANT COMMENT DESC RULE
==================================================

Comment Desc must state:
"Issued pursuant to section 320.19, Florida Statutes."

==================================================
VERY IMPORTANT PRIORITY CHANGE RULE
==================================================

Tax lien becomes:
First lien.

Existing first lien:
Becomes:
Second lien.

==================================================
VERY IMPORTANT ORIGINAL DOCUMENT RULE
==================================================

Department requires:
- Original HSMV 82139
- Signed certified receipt card

==================================================
VERY IMPORTANT PRIOR LIENHOLDER RULE
==================================================

Prior first lienholder:
Must still submit:
Required documents:
To FLHSMV.

==================================================
MAILING ADDRESS RULE
==================================================

Send to:

Department of Highway Safety & Motor Vehicles
2900 Apalachee Parkway, MS #72
Tallahassee, Florida 32399-0500

==================================================
VERY IMPORTANT TITLE SUBMISSION RULE
==================================================

Prior lienholder submits:
- Florida Certificate of Title
- Copy of HSMV 82139

==================================================
VERY IMPORTANT COPY RULE
==================================================

Copy of HSMV 82139:
Helps FLHSMV determine:
Where to direct title.

==================================================
VERY IMPORTANT ELT RULE
==================================================

Submission requirement:
Does NOT apply:
To ELT lienholders.

==================================================
FILING TAX LIEN — NO PRIMARY LIEN EXISTS
==================================================

If no primary lien:
Simplified process applies.

==================================================
REQUIRED DOCUMENTS — NO PRIOR LIEN
==================================================

Required:
1. HSMV 82139
2. Title correction
3. Original HSMV 82139
4. Title if available

==================================================
VERY IMPORTANT HSMV 82139 SECTION RULE
==================================================

If no prior lien:
Complete:
Sections 1 and 2 only.

==================================================
VERY IMPORTANT FIRST LIEN RULE
==================================================

Tax lien added as:
First lien.

==================================================
VERY IMPORTANT TITLE SUBMISSION TIMING RULE
==================================================

If owner submits title:
Before paperwork:
Send title with paperwork.

==================================================
VERY IMPORTANT SHRED RULE
==================================================

If title received:
After paperwork already sent:
Agency should:
Shred title.

==================================================
REMOVING TAX LIENS
==================================================

Removal handled under:
- TL-33
