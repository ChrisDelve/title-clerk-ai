TITLE:\
FLHSMV TL-22 \'97 Assembled From Parts and Kit Vehicle Procedures\
\
CATEGORY:\
Assembled Vehicles\
Kit Cars\
Custom Builds\
VIN Verification\
Special Construction Vehicles\
\
--------------------------------------------------\
\
RULE:\
Vehicles assembled from parts or constructed from kits require enhanced verification and inspection procedures before title issuance.\
\
VERIFY:\
- VIN assignment/status\
- Supporting ownership documents\
- Component receipts/documentation\
- Inspection requirements\
- Vehicle classification\
- Builder/applicant information\
\
ESCALATE IF:\
- VIN discrepancy exists\
- Missing component documentation\
- Fraud concerns arise\
- Vehicle classification unclear\
\
\
--------------------------------------------------\
\
RULE:\
Assembled-from-parts vehicles may require state-assigned VIN procedures before title issuance.\
\
VERIFY:\
- Existing VIN\
- Assigned VIN documentation\
- Inspection paperwork\
- Vehicle identification consistency\
\
ESCALATE IF:\
- VIN missing\
- Multiple VINs found\
- Altered VIN suspected\
- VIN assignment unclear\
\
\
--------------------------------------------------\
\
RULE:\
Major component parts documentation must support ownership and legality of vehicle construction.\
\
VERIFY:\
- Receipts for major components\
- Ownership documents\
- Supporting build documentation\
- Seller information if applicable\
\
COMMON MISTAKE:\
Missing receipts for major components during assembled vehicle processing.\
\
\
--------------------------------------------------\
\
RULE:\
Inspection requirements must be completed before title issuance for assembled vehicles.\
\
VERIFY:\
- Inspection completed\
- Supporting inspection paperwork\
- VIN verification\
- Vehicle roadworthiness requirements if applicable\
\
ESCALATE IF:\
- Inspection incomplete\
- VIN discrepancy exists\
- Vehicle configuration unclear\
\
\
--------------------------------------------------\
\
RULE:\
Kit vehicles and custom-built vehicles may require special body type/classification review.\
\
VERIFY:\
- Vehicle classification\
- Body type\
- Weight if applicable\
- Supporting build specifications\
\
ESCALATE IF:\
- Classification unclear\
- Vehicle category uncertain\
- Weight/body type conflict exists\
\
\
--------------------------------------------------\
\
RULE:\
Ownership documentation must establish legal ownership of vehicle components and completed vehicle.\
\
VERIFY:\
- Builder ownership\
- Component ownership\
- Bills of sale\
- Supporting title records if applicable\
\
ESCALATE IF:\
- Ownership conflict exists\
- Missing ownership records\
- Suspicious component history\
\
\
--------------------------------------------------\
\
RULE:\
Lienholder information must be verified if financing exists on assembled vehicle transaction.\
\
VERIFY:\
- Lienholder name\
- ELT status if applicable\
- Supporting lender documentation\
- Vehicle identification consistency\
\
ESCALATE IF:\
- Lender information conflict exists\
- VIN mismatch affects lien records\
- Missing lender documentation\
\
\
--------------------------------------------------\
\
RULE:\
Out-of-state assembled or specially constructed vehicles may require additional review before Florida title issuance.\
\
VERIFY:\
- Prior state title/registration\
- Existing branding/classification\
- Inspection requirements\
- VIN consistency\
\
ESCALATE IF:\
- Prior state title conflict exists\
- Classification unclear\
- Inspection requirements unresolved\
\
\
--------------------------------------------------\
\
RULE:\
Fraud prevention and theft verification are critical during assembled vehicle processing.\
\
VERIFY:\
- Component ownership\
- VIN legitimacy\
- Supporting receipts\
- Vehicle history if available\
\
ESCALATE IF:\
- Suspicious documentation\
- Altered VIN\
- Stolen parts concerns\
- Inconsistent ownership chain\
\
\
--------------------------------------------------\
\
COMMON DEALERSHIP/PROCESSING MISTAKES:\
- Missing component receipts\
- VIN mismatch\
- Missing inspection paperwork\
- Incorrect vehicle classification\
- Incomplete ownership documentation\
- Failure to verify assigned VIN\
- Missing builder information\
- Incorrect body type entry\
\
\
--------------------------------------------------\
\
HIGH-RISK ESCALATION SITUATIONS:\
- VIN discrepancies\
- Fraud concerns\
- Stolen parts concerns\
- Missing ownership documentation\
- Inspection irregularities\
- Vehicle classification conflicts\
- Altered paperwork\
- Multiple VIN issues\
- Out-of-state assembled vehicle conflicts\
\
\
--------------------------------------------------\
\
BEST PRACTICE:\
Before processing assembled-from-parts or kit vehicle:\
1. Verify VIN carefully.\
2. Verify all component ownership documentation.\
3. Verify inspection completion.\
4. Verify vehicle classification/body type.\
5. Verify ownership chain.\
6. Escalate any irregularity immediately.\
\
