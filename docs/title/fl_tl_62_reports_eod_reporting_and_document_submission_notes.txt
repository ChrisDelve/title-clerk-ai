==================================================
VERY IMPORTANT REPORT RULE
==================================================

Reports may be:
Generated multiple times:
During the same day.

==================================================
VERY IMPORTANT OPTIONAL REPORT RULE
==================================================

Some reports:
Optional to print:
And not required for FLHSMV submission.

==================================================
EOD REPORT RULES
==================================================

Every processing day:
EOD reports generated.

==================================================
VERY IMPORTANT DAILY REQUIREMENT RULE
==================================================

Agency MUST run:
Daily reports:
Even if there is:
NO activity.

==================================================
VERY IMPORTANT ATTESTATION RULE
==================================================

EOD reports contain:
- Attestation statement
- Signature requirement
- Date requirement

==================================================
REPORTS SENT TO FLHSMV
==================================================

Primary report:
- Transaction Summary Report

==================================================
TRANSACTION SUMMARY REPORT RULES
==================================================

Transaction Summary Report:
Shows:
- All transactions processed that day
- Counts
- Dollar totals
- County retained money
- Money due FLHSMV

==================================================
VERY IMPORTANT VEHICLE/VESSEL RULE
==================================================

Vehicles and vessels:
Appear separately:
On report.

Only:
ONE report:
Sent to FLHSMV.

==================================================
VERY IMPORTANT SUBMISSION DEADLINE RULE
==================================================

Reports and monies:
Must be submitted:
Within:
10 business days:
Of EOD process.

==================================================
VERY IMPORTANT DOCUMENT ORDER RULE
==================================================

Documents MUST remain:
In transaction ID order.

==================================================
VERY IMPORTANT BARCODE RULE
==================================================

Page one:
Transaction Summary Report with barcode:
Must be:
Top sheet.

==================================================
VERY IMPORTANT IMAGING ORDER RULE
==================================================

Documents submitted in this order:

1. Transaction Summary Report with barcode
2. Title transactions with HSMV 82041
3. HVUT scan cover sheets and docs
4. MVSR scan cover sheets and docs
5. MSWOL scan cover sheets and docs

==================================================
VERY IMPORTANT TITLE TRANSACTION RULE
==================================================

Title transaction packets include:
- Generated HSMV 82041
- Original supporting documents

==================================================
VERY IMPORTANT SCAN COVER SHEET RULE
==================================================

Scan cover sheets required for:
- HVUT
- MVSR
- MSWOL

==================================================
VERY IMPORTANT MAILING ORDER RULE
==================================================

When packaging multiple reports:
Reports must be:
In agency report number order.

==================================================
VERY IMPORTANT MAILING ADDRESS RULE
==================================================

Reports mailed to:

Florida Highway Safety and Motor Vehicles
Division of Motorist Services
High-Speed Scanning Unit
Clara H. Adams Building
2928 Apalachee Parkway, MS 71
Tallahassee, Florida 32399

==================================================
COUNTY RETAINED REPORTS
==================================================

Required retention:
Minimum:
1 year.

==================================================
VERY IMPORTANT RETENTION RULE
==================================================

Required EOD retained reports:
Must:
- Run daily
- Be retained minimum one year
- Be retained even with no activity

==================================================
SCAN EOD DETAIL REPORT
==================================================

Displays:
- Titles
- HVUT registrations
- Stops

Required for scanning.

==================================================
VERY IMPORTANT SCAN EOD RULE
==================================================

Scan EOD Detail:
NOT submitted:
To FLHSMV.

==================================================
TITLE AUDIT REPORT
==================================================

Tracks:
- Fast titles issued
- Fast titles voided

==================================================
VERY IMPORTANT TITLE AUDIT RULE
==================================================

Must balance:
To titles printed or voided:
That day.

==================================================
VERY IMPORTANT TITLE AUDIT RETENTION RULE
==================================================

Retain:
- Title Audit Report
- Supporting documents

For:
1 year.

Then:
Destroy.

==================================================
INITIAL EXEMPT REPORT
==================================================

Tracks:
Initial Registration Fee exemptions.

==================================================
VERY IMPORTANT INITIAL EXEMPT RULE
==================================================

Shows:
- User ID
- Exemption code used

==================================================
VERY IMPORTANT INITIAL EXEMPT DOCUMENT RULE
==================================================

Retain:
- Initial Exempt Report
- HSMV 82002
- Supporting documents

For:
1 year.

Then:
Destroy.

==================================================
REDUCED ANNUAL USE FEE REPORT
==================================================

Tracks:
Reduced fees for:
- Aquaculture plates
- Save Our Seas plates

==================================================
VERY IMPORTANT REDUCED FEE RULE
==================================================

Reduced fee allowed:
If:
10 or more plates:
Requested by same owner.

==================================================
VERY IMPORTANT REDUCED FEE RETENTION RULE
==================================================

Retain:
Report and supporting docs:
1 year.

Then:
Destroy.

==================================================
HEAVY VEHICLE USE TAX REPORT
==================================================

Tracks:
HVUT registrations.

==================================================
VERY IMPORTANT HVUT RULE
==================================================

Included when:
HVUT question answered:
"Yes"

==================================================
VERY IMPORTANT HVUT ORIGINAL DOCUMENT RULE
==================================================

Original HVUT transactions:
Must be sent:
With Transaction Summary Report.

==================================================
VERY IMPORTANT HVUT ERROR RULE
==================================================

Motorist Review Error Reports:
Emailed monthly:
If errors found.

==================================================
VERY IMPORTANT HVUT RETENTION RULE
==================================================

Retain:
HVUT reports and supporting docs:
Minimum 1 year.

==================================================
VERY IMPORTANT DOCUMENT CONDITION RULE
==================================================

Before batching:
Check for:
- Torn paper
- Damaged paper
- Illegible printing

==================================================
VERY IMPORTANT REPAIR RULE
==================================================

Repair torn paper:
Using:
Scotch tape.

==================================================
VERY IMPORTANT FLATTEN RULE
==================================================

Documents:
Must be flattened.
No folds.

==================================================
VERY IMPORTANT POST-IT RULE
==================================================

Remove:
Post-its and notes:
Covering information.

==================================================
VERY IMPORTANT BARCODE LEGIBILITY RULE
==================================================

Barcodes must:
- Be legible
- Remain unobstructed

==================================================
VERY IMPORTANT STAPLE RULE
==================================================

No more than:
One staple:
Upper left corner.

==================================================
VERY IMPORTANT PACKAGING RULE
==================================================

Batch order:

1. Transaction Summary Report
2. HSMV 82041 forms
3. Supporting docs for HSMV 82041
4. 93011 forms
5. Supporting docs for 93011

==================================================
VERY IMPORTANT TRANSACTION ID RULE
==================================================

Documents:
Must remain:
In numerical order:
By transaction ID.

==================================================
VERY IMPORTANT SUPPORTING DOC RULE
==================================================

Do NOT send:
Supporting docs:
Without matching:
93011 or HSMV 82041.

==================================================
VERY IMPORTANT SHIPPING RULE
==================================================

Documents should:
Fit snugly:
Inside shipping box.

==================================================
VERY IMPORTANT STACK HEIGHT RULE
==================================================

Stacks:
No taller than:
4 inches.
