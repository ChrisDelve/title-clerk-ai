==================================================
TRI-VEHICLE DEFINITION
==================================================

Tri-vehicle:
An enclosed three-wheeled passenger vehicle.

==================================================
VERY IMPORTANT WHEEL RULE
==================================================

Tri-vehicle:
Must operate:
With three wheels:
In contact with ground.

==================================================
VERY IMPORTANT WEIGHT RULE
==================================================

Minimum unladen weight:
900 pounds.

==================================================
VERY IMPORTANT OCCUPANT RULE
==================================================

Must have:
Single completely enclosed:
Occupant compartment.

==================================================
VERY IMPORTANT PRODUCTION RULE
==================================================

Vehicle must be produced:
Minimum:
300 units:
Per calendar year.

==================================================
VERY IMPORTANT SPEED RULE
==================================================

Vehicle must be capable of:
Greater than:
60 MPH:
On level ground.

==================================================
VERY IMPORTANT SEAT RULE
==================================================

Seats must comply with:
FMVSS 207
Seating Systems.

==================================================
VERY IMPORTANT STEERING RULE
==================================================

Vehicle must have:
Steering wheel.

==================================================
VERY IMPORTANT PROPULSION RULE
==================================================

Propulsion unit:
Must be located:
Forward or aft:
Of occupant compartment.

==================================================
VERY IMPORTANT SEAT BELT RULE
==================================================

Seat belts must comply with:
FMVSS 209.

==================================================
VERY IMPORTANT WINDSHIELD RULE
==================================================

Windshield must comply with:
FMVSS 205.

==================================================
VERY IMPORTANT WIPER RULE
==================================================

Wiper and washer system:
Must comply with:
FMVSS 104.

==================================================
VERY IMPORTANT ROLLOVER RULE
==================================================

Vehicle structure:
Must comply with:
FMVSS 216
Rollover Crush Resistance.

==================================================
VERY IMPORTANT ILEV RULE
==================================================

Tri-vehicles:
Are inherently low-emission vehicles (ILEV).

==================================================
VERY IMPORTANT HOV RULE
==================================================

Tri-vehicles may:
Operate in:
HOV lanes.

==================================================
VERY IMPORTANT BODY TYPE RULE
==================================================

Tri-vehicle body type:
3W.

==================================================
VERY IMPORTANT DESCRIPTION RULE
==================================================

Vehicle description:
Tri-vehicle.

==================================================
VERY IMPORTANT REGISTRATION RULE
==================================================

Tri-vehicles:
Registered and taxed:
Like automobiles.

==================================================
VERY IMPORTANT FEE RULE
==================================================

Fees calculated:
Like regular automobiles:
Using:
- Net weight
- Registered use
- Appropriate class code

==================================================
VERY IMPORTANT MOTORCYCLE RULE
==================================================

Tri-vehicle:
Is NOT a motorcycle.

==================================================
VERY IMPORTANT FRVIS RULE
==================================================

FRVIS setup:
Vehicle Type = AU
Body Type = 3W

==================================================
VERY IMPORTANT REG USE RULE
==================================================

Proper registration use code:
Must be entered.

==================================================
VERY IMPORTANT MC BODY TYPE RULE
==================================================

FRVIS will NOT allow:
Tri-vehicle:
As motorcycle body type:
MC.

==================================================
VERY IMPORTANT DEALER LICENSE RULE
==================================================

Manufacturers, importers,
distributors, and dealers:
Must be licensed.

==================================================
VERY IMPORTANT DEALER SERVICES RULE
==================================================

Dealer Services:
Verifies applications:
With NHTSA.

==================================================
VERY IMPORTANT NHTSA RULE
==================================================

Only NHTSA-approved tri-vehicles:
May be titled and registered.

==================================================
VERY IMPORTANT AUTHORIZED SOURCE RULE
==================================================

Tri-vehicles must originate from:
Authorized:
- Manufacturers
- Importers
- Distributors

Insurance requirements:

VERY IMPORTANT CORE CONCEPT
==================================================

Tri-vehicles:
Are legally treated:
As enclosed automobile-type vehicles,
NOT motorcycles,
Even though they operate with:
Three wheels.
