TITLE:\
FLHSMV TL-05 \'97 Duplicate Title Procedures\
\
CATEGORY:\
Duplicate Titles\
Lost Titles\
Electronic Titles\
Lienholder Verification\
Ownership Verification\
\
--------------------------------------------------\
\
RULE:\
A duplicate title may be requested if the original title is:\
- Lost\
- Stolen\
- Destroyed\
- Unavailable\
\
VERIFY:\
- Current ownership\
- Title status\
- Lienholder information\
- Customer identification\
- VIN accuracy\
\
ESCALATE IF:\
- Ownership mismatch exists\
- Title history unclear\
- Fraud concerns arise\
\
\
--------------------------------------------------\
\
RULE:\
Duplicate title applications require ownership verification.\
\
VERIFY:\
- Correct owner names\
- Signature requirements\
- Identification requirements\
- Title application completion\
\
COMMON MISTAKE:\
Submitting duplicate title paperwork with ownership mismatch.\
\
\
--------------------------------------------------\
\
RULE:\
If title ownership conjunction states AND:\
\uc0\u8594  All owners must sign duplicate title paperwork.\
\
RULE:\
If ownership conjunction states OR:\
\uc0\u8594  One owner may sign.\
\
VERIFY:\
- Ownership wording on title record\
- Required signatures present\
\
ESCALATE IF:\
- Ownership wording unclear\
- Deceased owner involved\
- POA involved\
\
\
--------------------------------------------------\
\
RULE:\
Electronic titles may require lienholder involvement before duplicate title issuance.\
\
VERIFY:\
- ELT status\
- Active lienholder\
- Lien satisfaction status\
- Bank information\
\
ESCALATE IF:\
- Lien release missing\
- ELT status unclear\
- Lienholder mismatch exists\
\
\
--------------------------------------------------\
\
RULE:\
Duplicate title processing may differ depending on whether title is:\
- Paper\
- Electronic\
- Out-of-state\
\
VERIFY:\
- State of title origin\
- Current title status\
- Supporting documentation\
\
ESCALATE IF:\
- Out-of-state title complications exist\
- Prior title unresolved\
- Duplicate ownership records appear\
\
\
--------------------------------------------------\
\
RULE:\
Customer identification must satisfy FLHSMV identification requirements.\
\
VERIFY:\
- Name matches ownership\
- Valid identification\
- Supporting ID copy if required\
\
ESCALATE IF:\
- Name mismatch exists\
- Identity unclear\
- Fraud concerns arise\
\
\
--------------------------------------------------\
\
RULE:\
VIN must match all title and supporting records.\
\
VERIFY:\
- Title application VIN\
- System VIN\
- Existing title VIN\
\
ESCALATE IF:\
- Any VIN discrepancy exists\
- Altered VIN suspected\
- VIN unreadable\
\
\
--------------------------------------------------\
\
RULE:\
POA documentation may be acceptable for duplicate title processing if properly executed.\
\
VERIFY:\
- POA completeness\
- Authorized signer\
- Supporting documentation\
\
ESCALATE IF:\
- POA altered\
- Authority unclear\
- Signature mismatch\
\
\
--------------------------------------------------\
\
COMMON DEALERSHIP MISTAKES:\
- Missing co-owner signature\
- Incorrect ownership conjunction interpretation\
- Duplicate title requested before lien release\
- Incorrect ELT assumptions\
- Missing ID documentation\
- VIN mismatch overlooked\
- Missing POA paperwork\
- Submitting incomplete title application\
\
\
--------------------------------------------------\
\
HIGH-RISK ESCALATION SITUATIONS:\
- Ownership disputes\
- Active lien complications\
- Out-of-state duplicate title issues\
- Fraud concerns\
- VIN discrepancies\
- Deceased owner\
- Trust/entity ownership\
- Title history inconsistencies\
\
\
--------------------------------------------------\
\
BEST PRACTICE:\
Before processing duplicate title paperwork:\
1. Verify title status.\
2. Verify lienholder status.\
3. Verify ownership conjunction.\
4. Verify signatures.\
5. Confirm VIN consistency.\
\
